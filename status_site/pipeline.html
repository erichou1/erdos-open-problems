01

What this pipeline does

STAGE 1 · ADAPT

The fixed meta-prompt is adapted to each problem

A fresh chat receives the template prompt plus an instruction to adapt it to the target problem and its literature, and returns a problem-specific research prompt.

STAGE 2 · RESEARCH

One chat per problem

The adapted prompt is submitted in a second chat (named after the problem) so the whole search — and any later manual follow-up — lives in one conversation.

STAGE 3 · CONTINUE

Nudged toward a complete unconditional result

While an attempt does not declare a complete result, a continuation nudge is sent in the same chat, up to the configured round cap.

TRANSPARENCY

Every stage of iteration is shown

Each adaptation, research attempt, and continuation round is recorded with a link to the exact ChatGPT conversation. Text is truncated here; open the chat for the full transcript.
