What the labels mean

Working now

A worker currently holds this problem and is asking ChatGPT, checking sources, or testing a proof.

Research attempt

One complete or interrupted pass through several possible approaches to a problem.

Math statement proposed

A lemma or intermediate claim suggested by a model. It is not accepted as true yet.

Checked evidence

A computation, source, or proof that passed the pipeline’s independent validation rules.

ChatGPT exchange

One prompt and response. New exchanges save the exact chat URL; older ones may have only tamper-evident hashes.

Aristotle proof draft

Lean code generated by Aristotle. It remains a draft until the local Lean kernel and statement-matching checks pass.
