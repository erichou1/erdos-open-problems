\noindent\textbf{Status:} OPEN \quad \textbf{No prize}

\noindent\textbf{Tags:} number theory, primes

\medskip
\noindent\textbf{Problem Statement:}

Let $C>1$. Does the set of integers of the form $p+\lfloor C^k\rfloor$, for some prime $p$ and $k\geq 0$, have density $>0$?

\bigskip
\noindent\textbf{Remarks:}

Originally asked to Erd\H{o}s by Kalm\'{a}r. Erd\H{o}s believed the answer is yes. Romanoff \cite{Ro34} proved that the answer is yes if $C$ is an integer.

Ding \cite{Di25} has proved that this is true for almost all $C>1$.

\bigskip
\noindent\textbf{References:}

[Di25] Y. Ding, On a Romanoff type problem of Erd\H{o}s and Kalm\'{a}r. arXiv:2503.22700 (2025).
