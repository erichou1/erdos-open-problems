OPEN This is open, and cannot be resolved with a finite computation.
For which integers $a\geq 1$ and primes $p$ is there a finite upper bound on those $k$ such that there are $a=a_1<\cdots<a_n$ with\[p^k \mid (a_1!+\cdots+a_n!)?\]If $f(a,p)$ is the greatest such $k$, how does this function behave?

Is there a prime $p$ and an infinite sequence $a_1<a_2<\cdots$ such that if $p^{m_k}$ is the highest power of $p$ dividing $\sum_{i\leq k}a_i!$ then $m_k\to \infty$?
#404: [ErGr80,p.79]

  • I made an attempt repository for exact values in the $p=2$ column of Problem #404: $f(a,2)\quad(1\le a\le256).$
    Repository: https://github.com/KitaKen1/erdos-404-p2-landscape
    Visual table: https://kitaken1.github.io/erdos-404-p2-landscape/

    In particular: $f(2,2)=254.$ This is worth singling out because the Problem #404 page records Lin's upper bound $f(2,2)\le254$; the repository gives the matching lower-bound witness and a two-sided Lean/lean4web check.
    Lean4web for $f(2,2)=254$: https://live.lean-lang.org/#url=https%3A%2F%2Fraw.githubusercontent.com%2FKitaKen1%2Ferdos-404-p2-landscape%2Fmain%2Flean%2FS404_lean4web_2_2.lean

    Here $v_2(m)$ denotes the exponent of $2$ in $m$. The lower-bound side is just an explicit increasing list whose factorial sum is divisible by the claimed power of $2$. The upper-bound side is finite: by Legendre's formula,$v_2(n!)=\lfloor n/2\rfloor+\lfloor n/4\rfloor+\lfloor n/8\rfloor+\cdots,$ so to rule out exponent $K+1$, search modulo $2^{K+1}$, and once $v_2(n!)\ge K+1$, all later factorials are $0$ modulo $2^{K+1}$.

    For odd $a$, this immediately freezes the value at $f(a,2)=v_2(a!)$. For even $a$, cancellations make the values much wilder.

    The repository also records a few observations from the table. For example, the largest value found in $1\le a\le256$ is $f(34,2)=18444.$

  • It seems to me that $f(1,5)$ is already extremely large, and possibly even infinite; for each $m$, it appears that there are lots of ways to recursively find a set of factorials up to $(5m-1)!$ and starting at $1!$ that sum to something that divides at least as many powers of $5$ as $5!$. For instance
    $$ 1! + 4! = 100_5$$
    $$1!+4!+10!+11! = 42121430000_5$$
    $$1! + 4! + 10! + 11! + 20! + 21! + 23! = 102401442411110300413412001000000_5$$
    Things get a little tricky every time one reaches $5m$ is a multiple of $5^2$ or higher powers, but there is a combinatorial explosion of possible cases and it seems very likely that one could continue this sort of process for quite a while.

    • The following lemma is obvious but potentially useful in iteratively lower bounding $f(a,p)$: let $P(a,p,N)$ denote the assertion that $p^{\nu_p(N!)} | a_1! + \dots + a_n!$ for some $a=a_1 < a_2 < \dots < a_n < N$. Then, if the $2^H$ subset sums of $N!/p^{\nu_p(N!)}, \dots, (N+H-1)!/p^{\nu_p(N!)}$ cover all the residue classes of ${\bf Z}/p^{\nu_p((N+H)!/N!)}{\bf Z}$, $P(a,p,N)$ implies $P(a,p,N+H)$. For instance this shows that $P(a,p,N)$ implies $P(a,p,N+1)$ whenever $N+1$ is not divisible by $p$, and for many $p$ (e.g., $p=5$) it should be possible to show computationally that $P(a,p,N)$ implies $P(a,p,N+p)$ whenever $N+p$ is divisible by exactly one power of $p$ by showing that the $2^p$ subset sums of $0!, 1!, \dots, (p-1)!$ cover all of ${\bf Z}/p{\bf Z}$. More generally, some finite computation seems to let one show that $P(a,p,N)$ implies $P(a,p,N+p^j)$ whenever $N+p^j$ is divisible by exactly $j$ powers of $p$ for any fixed $j$ (with the $N=0$ case being key); doing this for $j \leq J$ should already get some lower bound of the shape $f(a,p) \geq \nu_p( (p^{J+1}-1)! )$.

  • I do not understand the first two questions. The first one makes no sense: the uniform bound on $f(a,p)$ is not possible, since we can find $k$ as large as we want if we allow $a$ to be sufficiently large. Taking a look into the Graham-Erdos problem book (p. 79), I also see that it is not what they wanted: the power of $p$ in $p^k$ and the number of summands in $a_1!+\cdots+a_k!$ were not supposed to be the same.

    To make changes minimal I suggest to replace the first half of the problem (i.e., the first two questions) with:


    For which positive integers $a$ and which primes $p$ there exists a finite least upper bound $f(a,p)$ on the exponents $m$ such that\[ p^m | (a_1!+\cdots+a_k!) \]for some $k\geq1$ and some integers $a=a_1<\cdots< a_k$?


    The comment on Lin's result should then be $f(2,2)=254$. The third question seems to be ok.
