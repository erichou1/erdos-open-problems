Let $n\geq 2$ and $\pi(n)<k\leq n$. Let $f(k,n)$ be the smallest integer $r$ such that in any $A\subseteq \{1,\ldots,n\}$ of size $\lvert A\rvert=k$ there exist primes $p_1,\ldots,p_r$ such that $>r$ many $a\in A$ are only divisible by primes from $\{p_1,\ldots,p_r\}$.

Is it true that\[2\pi(n^{1/2})-f(\pi(n)+1,n)\to \infty\]as $n\to \infty$?

In general, estimate $f(k,n)$, particularly when $\pi(n)+1<k=o(n)$.
#983: [Er70b,p.138]
number theory

It is trivial that $f(k,n)\leq \pi(n)$. Erdős and Straus [Er70b] proved that\[f(\pi(n)+1,n)=2\pi(n^{1/2})+o_A\left(\frac{n^{1/2}}{(\log n)^A}\right)\]for any $A>0$ and, for any constant $1>c>0$,\[f(cn,n)=\log\log n+(c_1+o(1))\sqrt{2\log\log n},\]where\[c=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^{c_1}e^{-x^2/2}\mathrm{d}x.\]

    • I have looked into this.

      It turns out this is an *improvement* to Erdos' own construction/argument, which the original is too weak to resolve the conjecture. The relationship here is interesting.

      First, this can be formulated via Hall's theorem as
      $$ f(k, n) := \max_{A \subseteq [n], |A| = k} \min_{S\text{ set of primes}} \{ |S|: |A[S]| > |S| \}, $$
      where $A[S]$ is the set of $a \in A$ with prime divisors only in $S$. To do this, we construct a bipartite graph with vertices being disjoint union of elements of $A$ and primes, joining $a \in A$ to its prime divisors. If $k > \pi(n)$, there can't be a perfect matching, and we apply Hall that $S$ must exist.

      Erdos shows that
      $$ f(k, n) \le 2\pi(\sqrt{n}) + 1. $$
      This part is reproved by Price. Basically, for $B \subseteq A$ let $P(B)$ be the set of prime divisors of $B$. Choose $B$ minimal by inclusion so that $|B| > |P(B)|$. By considering $B \setminus \{x\}$ for $x \in B$, we can deduce $|P(B)| = |B| - 1$ and that every $p \in P(B)$ divides at least $2$ elements of $B$. And each element of $B$ is divisible by at most $1$ prime $> \sqrt{n}$. Hence $|P(B)| \le \pi(\sqrt{n}) + |B|/2$. This gives the claim.

      Now Erdos' main construction is a "semiprime polygon" where $pq$ means an edge between primes $p$ and $q$. And these edges are joined into a loop. This allows him to conclude the asymptotically sharp bound
      $$ f(\pi(n) + 1, n) \sim 2\pi(\sqrt{n}). $$
      However, Erdos also thinks that $2\pi(\sqrt{n}) - f(\pi(n) + 1, n) \to \infty$, which is the first question.

      It turns out that this is not the case and that you can have
      $$ f(\pi(n) + 1, n) = 2\pi(\sqrt{n}) + 1 $$
      infinitely often. This is by Price's construction and it basically is an improvement of Erdos' construction.

      It is known that there are infinitely many primes $p_N$ such that $p_{N-i} p_{N+i} < p_N^2$ for $1 \le i < N$. Choose $n := p_N^2 - 1$. Then $\pi(\sqrt{n}) = N-1.$ Consider the *path*
      $$ p_{2N-1}, p_1, p_{2N-2}, p_2, \ldots, p_{N+1}, p_{N-1}, p_N. $$
      We reuse Erdos' interpretation where a semiprime is an ordinary edge. We also let a prime correspond to a loop at a vertex. Then consider the set $C$ of this path plus the two endpoint loops, i.e.
      $$ C := \{ p_{2N-1}, p_{2N-1}p_1, \dots, p_{N-1}p_N, p_N \}. $$
      This kind of construction ensures that $|C| > |P(C)|$. Note that the inequality condition on the primes is to make sure that $C \subseteq [n]$. Now let $A$ be the union of $C$ and the set of primes $\le n$ that are not in $C$. It is seen that $|A| = \pi(n) + 1$. It is now not difficult to check that the $\min$ part of the definition of $f(k, n)$ above corresponding to this $A$ is indeed $2N-1$, which is what needs to be shown.

  • Thank you Woett for the detailed clarification on the "strictly more than $r$" vs "at least $r$" distinction. That's an important correction to the problem statement.

    I'd like to raise a related question about the quantifier structure of $f(k,n)$ that came up during computational investigation (discussion at github.com/teorth/erdosproblems#216).

    The current problem statement reads:

    $f(k,n)$ be the smallest integer $r$ such that in any $A \subseteq {1,\ldots,n}$ of size $|A|=k$ there exist primes $p_1,\ldots,p_r$ such that [more than] $r$ many $a \in A$ are only divisible by primes from ${p_1,\ldots,p_r}$.

    This "for any $A$, there exist primes" formulation suggests an adversarial/worst-case problem: the set $A$ is chosen first (adversarially), then we find covering primes.

    Question: Is this the correct quantifier order from [Er70b]?

    In the GitHub discussion, it was suggested that Erdős-Straus may have defined $f(k,n)$ with reversed quantifiers:

    "there exist primes $p_1,\ldots,p_r$ such that at least $k$ integers $m \leq n$ have all prime divisors contained in ${p_1,\ldots,p_r}$"

    Under this alternative interpretation, primes are chosen first to maximize coverage, making small primes $\leq \sqrt{n}$ optimal (each generates many smooth composites), naturally yielding $f \approx 2\pi(\sqrt{n})$.

    Under the adversarial interpretation currently stated, computational testing suggests $f$ is much smaller. For example, with $k = \pi(n)+1$:

    Any set $A$ of squarefree composites has "triangle" structure: three primes $p,q,r$ cover the semiprimes $pq, pr, qr$ (if present), giving efficient coverage
    Testing up to $n = 10{,}000$ consistently yields small values that don't approach $2\pi(\sqrt{n})$
    Could someone with access to [Er70b] ("Some applications of graph theory to number theory", 1969) confirm the exact quantifier structure in the original definition?

    Woett's lower bound construction (using $A_0$ with paired products plus primes) is illuminating, but I want to ensure I'm understanding the correct problem before analyzing it further.

  • Let $2 = p_1 < p_2 < \cdots < p_t$ be those primes smaller than or equal to $(2 - \epsilon)\sqrt{n}$. Erdős and Straus then construct a set $A_0$ of size $t$, where $A_0$ consists of certain pairs of products $p_ip_j < n$ such that every $p_i$ (with $i \le t$) occurs in exactly two products. Finishing up, they let $A$ (with $k = |A| = \pi(n)+1$) be the union $$A := A_0 \cup \{2p_{t+1}, 3p_{t+1}\} \cup \bigcup_{i = t+2}^{\pi(n)} p_i.$$For this set $A$ one can then check that $r = t+1$, as $\{p_1, p_2, \ldots, p_{t+1}\}$ is the smallest set of primes such that there are more than $r$ elements $a \in A$ whose prime decompositions only contain $p_1, \ldots, p_{r}$. Hence, we get the lower bound $f(\pi(n)+1, n) > t =\pi((2-\epsilon)\sqrt{n})$.

    Now note that, for fixed $n$, $f(k, n)$ is a non-increasing function of $k$. Since Erdős and Straus also showed $f(\pi(n)+1,n) \le 2\pi(\sqrt{n}) + 1$, this gives $f(k, n) \le 2\pi(\sqrt{n}) + 1$ for all $k > \pi(n)$. On the other hand, we could have slightly tweaked the lower bound construction we just saw as follows:

    Let $A_0$ still consist of $t$ pairs of products $p_ip_j < n$ as explained in [Er70b], but as opposed to using the primes $p_1, p_2, \ldots, p_t$, we use $p_2, p_3, \ldots, p_{t+1}$ instead. We then set $$A := A_0 \cup \bigcup_{i=2}^{\pi(\frac{1}{2}n)} \{2p_i\} \cup \bigcup_{i = t+2}^{\pi(n)} p_i.$$In this case we still get $r = t+1$, while $k = \pi(n) + \pi(\frac{1}{2}n) - 2$. Combining upper and lower bound and using the prime number theorem we deduce $f(k,n) = (4 + o(1))\frac{n^{1/2}}{\log n}$ for all $k$ with $\pi(n) < k < (\frac{3}{2} + o(1))\frac{n}{\log n}$. One can probably increase the range of $k$ for which this estimate holds by tweaking $A$ further.

    Based on the proven value of $f(cn, n)$, presumably some kind of phase-shift occurs somewhere, if we let $k$ increase from $\frac{cn}{\log n}$ to $cn$.

    Finally, $f(k,n)$ might still be interesting for the range $(1-o(1))n < k < n$. To give two quick examples, if $k > n - \pi(\sqrt{n})$, then we are guaranteed to have $\{p, p^2\} \subset A$ (so that $f(k,n) = 1$), and if $k > n - \pi(\frac{1}{4}n)$ we are guaranteed to have $\{p, 2p, 4p\} \subset A$ (so that $f(k,n) \le 2$). One can wonder what the slowest growest function $g(n)$ is such that $k < n - g(n)$ implies $f(k,n) \rightarrow \infty$. By the value of $f(cn, n)$ we know $g(n) = o(n)$, while generalizing the second example we just gave leads (I think) to $g(n) > (1 - o(1))\pi(n)$.
