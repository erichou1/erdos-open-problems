FALSIFIABLE Open, but could be disproved with a finite counterexample.
Let $A$ be a finite set and\[B=\{ n \geq 1 : a\mid n\textrm{ for some }a\in A\}.\]Is it true that, for every $m>n\geq \max(A)$,\[\frac{\lvert B\cap [1,m]\rvert }{m}< 2\frac{\lvert B\cap [1,n]\rvert}{n}?\]
#488: [Er61,p.236][Er66,p.150][Er80,p.112]
number theory

There are no solutions, partial or complete, claimed in the comments.
The constant $2$ would be the best possible here, as witnessed by taking $A=\{a\}$, $n=2a-1$, and $m=2a$.

This problem is also discussed in problem E5 of Guy's collection [Gu04].

In [Er61] this problem is as stated above, but with $a\mid n$ in the definition of $B$ replaced by $a\nmid n$. This is most likely a typo (especially since the problem is also given as stated above in [Er66]). There have been several counterexamples given for this alternate problem. Cambie has observed that, if $A$ is the set of primes bounded above by $n$, and $m=2n$, then\[\frac{\lvert B\cap [1,m]\rvert }{m}=\frac{\pi(2n)-\pi(n)+1}{2n}\sim \frac{1}{2\log n}\]while\[\frac{\lvert B\cap [1,n]\rvert}{n}=\frac{1}{n}.\]Further concrete counterexamples, found by Alexeev and Aristotle, are given in the comments section.

  • Here is a small observation on the corrected “multiples” version of #488. It may already be known, but I did not see it stated explicitly in the thread.

    Let\[
    F_A(x)=|\{1\le t\le x:\ a\mid t\text{ for some }a\in A\}|.
    \]I claim that the conjectured inequality holds when \(A\) has two elements.

    Write \(A=\{a,b\}\), where \(2\le a<b\). First suppose \(a\nmid b\). Then \(b\) is counted and is not a multiple of \(a\), so for every \(n\ge b\),\[
    F_A(n)\ge \lfloor n/a\rfloor+1\ge (n+1)/a.
    \]On the other hand \(F_A(m)\le m/a+m/b<2m/a\), and hence\[
    \frac{F_A(m)}{m}<\frac{2}{a}\le \frac{2F_A(n)}{n+1}<\frac{2F_A(n)}{n}.
    \]If \(a\mid b\), then the counted set is just the multiples of \(a\), so \(F_A(x)=\lfloor x/a\rfloor\). Since \(n\ge b\ge 2a\), we have \(\lfloor n/a\rfloor\ge2\). Thus\[
    2mF_A(n)-nF_A(m)
    \ge m\left(2\lfloor n/a\rfloor-\frac{n}{a}\right)>0,
    \]which gives the desired strict inequality.

    So #488 holds for every two-element set \(A\).

    The near-sharp examples are already visible here: for \(A=\{a,a+1\}\), \(n=2a-1\), and \(m=a^2\),\[
    \frac{F_A(m)/m}{F_A(n)/n}
    =
    \frac{(2a-1)^2}{2a^2},
    \]which tends to \(2\) from below. This leaves the first genuinely nontrivial case at primitive sets of size at least \(3\), where the overlaps between multiple sets start to matter.

  • One notable thing about this problem - unusual for an Erdos problem - is that there are several ways to make the problem easier to prove, yet the problem remains difficult. For instance, here are four ways to reduce the difficulty level:

    1. Take the asymptotic limit $m \to \infty$.
    2. Take the asymptotic limit $\max(A) = o(n)$ (and perhaps replace $2$ by $(2+o(1))$ to be able to take advantage of this).
    3. Replace the factor $2$ by an unspecified constant $C$.
    4. Assume that $A$ consists entirely of primes.
    What I find remarkable is that even when enabling multiple of these "cheats", the problem is still nontrivial. I can resolve the problem with Cheats 1, 3, and 4 enabled: that is to say I can show that
    $$ \lim_{m \to \infty} \frac{|B \cap [1,m]|}{m} \ll \frac{|B \cap [1,n]}{n}$$
    when $A$ consists entirely of primes. Here the left-hand side is $1 - \prod_{p \in A} (1-\frac{1}{p})$. When $\sum_p \frac{1}{p}$ is small the right-hand side is $\asymp \sum_p \frac{1}{p}$ by the Bonferroni inequalities, and then when $\sum_p \frac{1}{p}$ is large the RHS is $\asymp 1$ by monotonicity, so in either case thed claim follows. But I don't see a way to prove this version in a way that suggests how to remove more of the cheats.

  • Progress on EP-488: \(a=2\) case proved, Chojecki reduction fails for \(a\geq3\)

    Some results on the density-doubling conjecture, building on the pair-tail decomposition framework from Conjecture 4.8.

    1. The \(a=2\) case. For every primitive set \(A\) with \(2 \in A\) (equivalently, \(\min(A)=2\)), the conjecture holds.

    Since every element of \(A\) is \(\geq 2\), integer 1 is never counted, so \(F(m) \leq m-1\) and \(F(m)/m < 1\) for all \(m\).

    If \(A \neq \{2\}\), choose \(b \in A \setminus \{2\}\). Since \(A\) is primitive and \(2 \in A\), this \(b\) is necessarily odd. For \(n \geq b\):\[
    F(n) \geq \left\lfloor \frac{n}{2} \right\rfloor + \left\lfloor \frac{n}{b} \right\rfloor - \left\lfloor \frac{n}{2b} \right\rfloor = \left\lfloor \frac{n}{2} \right\rfloor + \#\{\text{odd multiples of } b \leq n\} > \frac{n}{2}.
    \]Therefore \(2F(n)/n > 1 > F(m)/m\).

    For the singleton \(A = \{2\}\): \(F(k) = \lfloor k/2 \rfloor\), so for all \(m > n \geq 2\),\[
    \frac{2F(n)}{n} = \frac{2\lfloor n/2 \rfloor}{n} > \frac{1}{2} \geq \frac{\lfloor m/2 \rfloor}{m} = \frac{F(m)}{m}.
    \]This covers all primitive sets with \(2 \in A\).

    **2. The Chojecki reduction fails for \(a \geq 3\).** The reduction to a fixed threshold \(s\) (first integer with \(F(s) \geq 5\)) does not extend to \(a \geq 3\). Counterexample family:

    For any \(a \geq 3\), take \(b = 5a+1\) and \(T = \{t \in \mathbb{Z} : 5a+1 < t \leq 10a+1,\; a \nmid t\}\). Then \(A = \{a,b\} \cup T\) is primitive: every element of \(\{b\} \cup T\) lies in \((5a,\, 10a+1]\), so each exceeds \((10a+1)/2\) and no two can divide each other; and \(a\) divides none of them by construction.

    The first \(s\) with \(F(s) \geq 5\) is \(s = 5a\), since below \(5a\), only multiples of \(a\) contribute and there are exactly 4 of them (\(a, 2a, 3a, 4a\)). So \(F(s) = 5\) and \(2F(s)/s = 2/a\).

    At \(m = 10a+1\): every integer in \([5a+1,\, 10a+1]\) is counted by \(A\) (divisible by \(a\) \(\to\) counted; not divisible by \(a\) \(\to\) in \(T\) or equals \(b\) \(\to\) counted). So \(F(10a+1) = 5 + (5a+1) = 5a+6\), giving\[
    \frac{F(m)}{m} = \frac{5a+6}{10a+1} > \frac{2}{a} = \frac{2F(s)}{s}
    \]since \(5a^2 - 14a - 2 > 0\) for all \(a \geq 3\).

    Concrete: \(a=3\), \(A = \{3,16,17,19,20,22,23,25,26,28,29,31\}\), \(F(31)/31 \approx 0.677 > 2/3\).

    Note: this does NOT disprove EP-488, since \(s = 5a < \max(A) = 10a+1\), so the counterexample lies outside the valid range \(n \geq \max(A)\).

    3. What remains. The original EP-488 statement (all \(m > n \geq \max(A)\)) is checked computationally across 25,000+ primitive systems with zero failures and margins \(\geq 33\%\) (defined as \((2F(n)/n - F(m)/m) / (2F(n)/n)\) for the tightest \((n,m)\) pair). The reduction to sieve oscillation still works within the \(n \geq \max(A)\) regime, where the key inequality becomes: for any finite \(Q \subset \mathbb{Z}_{\geq 2}\) and \(x \geq \max(Q)\), is \(A_Q(x) < 2\delta_Q x\)?

    For pairwise coprime \(Q\), Hall–Hildebrand type bounds on the complementary unsieved count operate at an \(e^\gamma\)-scale; translating that into the form \(A_Q(x) < 2\delta_Q x\) requires care. For non-coprime \(Q\), computational evidence shows strictly smaller oscillation — the lcm lattice produces 30–70% coefficient cancellation in the inclusion-exclusion.

  • The commentary gives an example where $A$ is a singleton of size around $n/2$, and the ratio is close to 2. There is a somewhat different example where $A$ now consists of numbers much smaller than $n$, and the ratio is about 1.03; nowhere near a counterexample, but still uniformly bounded away from 1, and indicative of the complexity of the problem.

    The counterexample comes from taking $A$ to be the primes between $n^{1/3}$ and $n^{1/2}$, and $m$ to be extremely large compared to $n$. The density at $m$ is then about $1 - \prod_{n^{1/3} < p < n^{1/2}} (1-\frac{1}{p})$, which by Mertens' theorem is about $1/3 = 0.333\dots$. The density at $n$ can be computed by inclusion-exclusion to be about $\sum_{n^{1/3} < p < n^{1/2}} 1/p - \sum_{n^{1/3} < p_1 < p_2 < n^{1/2}} 1/p_1 p_2$ which computes to $\log(3/2) - \frac{1}{2} \log^2(3/2) = 0.32326413$. The ratio here is $1.03114853\dots$.

    Some tweaking of this type of example seems to permit a slightly larger ratio than this, but it doesn't look like it gets close to 2. Proving this concretely looks tricky though. In the spirit of this paper of Granville-Soundararajan, the main task involves proving an inequality involving alternating sums of various integrals, which looks somewhat complicated.

    • Progress on Conjecture 4.8 (pair-tail split doubling) and a specific remaining gap

      Hi Mr. Chojecki,

      I've been working on the pair-tail split doubling conjecture (Conjecture 4.8 in your paper) using the quotient-tail framework from your Section 8, and I've made significant progress but hit a specific wall. I wanted to share what I have and ask whether you see a path through the remaining gap.

      What I've proved

      Starting from the quotient-tail decomposition
      $$F_{a,b\mid T}(x) = A_{Q_a}\!\left(\left\lfloor\frac{x}{a}\right\rfloor\right) + A_{Q_b}\!\left(\left\lfloor\frac{x}{b}\right\rfloor\right) - A_{Q_\ell}\!\left(\left\lfloor\frac{x}{\ell}\right\rfloor\right),$$
      I built a reduction chain:

      1. Exact periodicity. $F(n) = \delta n + c_{n \bmod P}$ with $\delta_{a,b\mid T} = \frac{\delta(Q_a)}{a} + \frac{\delta(Q_b)}{b} - \frac{\delta(Q_\ell)}{\ell} > 0$.

      2. Visible-slab reduction. The finite-window split inequality reduces to checks (V1), (V2), (V3) on the base visible slab $S_{\mathrm{vis}} = [N_0, \min(N_1, P-1)] \cap \mathbb{Z}$.

      3. Low-$F$ elimination. All visible-slab residues with $F(s) = 2$ or $F(s) = 3$ are automatically harmless (the threshold $2F(s)/s > 1/a + 1/b$ suffices). $F(s) = 4$ reduces to an explicit exceptional family, which I closed completely — including the sub-exceptional scaling family $(48k, 54k, \{64k, 72k, 81k\})$ via a scaling law: the quotient tails $\hat{Q}_{48k} = \hat{Q}_{54k} = \{3,4\}$ are $k$-independent, so $F_k(qk+t) = F_1(q)$, collapsing the infinite family to a finite base computation on 5103 residues.

      4. Window-bound theorem. For any window length $L$, $F(n) - F(n-L) \leq c(L)$ with $c(L)/L \to \delta$. This gives an inductive propagation principle that eliminates envelope comparisons beyond the visible slab.

      So the split inequality holds for all pair-tail systems where every visible-slab residue has $F(s) \leq 4$, plus the specific exceptional families at $F = 4$.

      The remaining gap: $F(s) \geq 5$ on the visible slab

      The endgame reduces to: for every visible-slab $s$ with $F(s) \geq 5$ and every $m > s$, prove
      $$\frac{F(m)}{m} < \frac{2F(s)}{s}.$$

      I've verified this computationally across tens of thousands of systems with zero counterexamples. The margins are consistently 30–50% of the slab ratio.

      What I've tried and what fails:

      - Universal bound $F(m)/m \leq \delta + W^+/m$ combined with $2F(s)/s \geq 10/s$: closes 98.6% of systems but fails when $3W^+ + |W^-| \geq 10$.

      - Per-residue bound $W^+ < 2F(s) - \delta s$: false. Counterexample at $(11, 15 \mid \{28, 31\})$, $s = 43$.

      - Combinatorial charging ($\Delta_a + \Delta_b \geq 2\Delta_\ell + 1$): the strong form is false.

      - Two-route via Proposition 5.1: killed by $G = $ first 14 primes at $n = 198$.

      The split inequality holds in every case, but every bound overshoots because the global $W^+$ is achieved at a different residue than the ratio peak $\sup_m F(m)/m$.

      The structural picture

      The inequality appears to be "soft" — it holds because the factor of 2 creates a multiplicative buffer that the periodic corrections cannot fully consume. At the ratio-maximizing $m^*$, the correction $c_{m^*}$ is always much less than $W^+$ (typically $c_{m^*}/W^+ \approx 0.3$–$0.5$). But I cannot prove this universally.

      Questions

      1. Do you see a way to prove $F(m)/m < 2F(s)/s$ for $F(s) \geq 5$ using the quotient-tail structure? Perhaps through the cluster expansion / Janson bound machinery from your Section 8?

      2. Is there a known technique for bounding $c_m/m$ (the ratio peak of periodic corrections) rather than $W^+/m$?

      3. Do you think Conjecture 4.8 is provable by this route, or does it require a fundamentally different approach for large tails?

      Happy to share the full proof chain, computational code, and counterexamples to the failed bounds.

    • Assuming the following inequality holds, this version of the problem is straightforward.

      For $A=\{a_1<\ldots<a_k\}\subset \mathbb{N}$ and an integer $n\geq a_k$, assume we have $$\sum_{\emptyset\neq I=\{a_{i_1},\ldots,a_{i_r}\}\subseteq A} (-1)^{r+1}\big\lfloor\frac{n}{lcm(a_{i_1},\ldots,a_{i_r})}\big\rfloor\geq -1+ \sum_{\emptyset\neq I=\{a_{i_1},\ldots,a_{i_r}\}\subseteq A}(-1)^{r+1} \frac{n}{lcm(a_{i_1},\ldots,a_{i_r})}.$$

      Then by the IEP, $B(n):=|B\cap[1,n]|=\sum_{\emptyset\neq I=\{a_{i_1},\ldots,a_{i_r}\}\subseteq A} (-1)^{r+1}\big\lfloor\frac{n}{lcm(a_{i_1},\ldots,a_{i_r})}\big\rfloor$ so for $m>n\geq a_k$, $$\frac{B(n)}{n}= \sum_{\emptyset\neq I=\{a_{i_1},\ldots,a_{i_r}\}\subseteq A} \frac{(-1)^{r+1}}{n}\big\lfloor\frac{n}{lcm(a_{i_1},\ldots,a_{i_r})}\big\rfloor \geq \frac{-1}{n}+ \sum_{\emptyset\neq I=\{a_{i_1},\ldots,a_{i_r}\}\subseteq A}(-1)^{r+1} \frac{1}{lcm(a_{i_1},\ldots,a_{i_r})}$$

      $$\geq \frac{-1}{n}+\sum_{\emptyset\neq I=\frac{-1}{n}+\{a_{i_1},\ldots,a_{i_r}\}\subseteq A} \frac{(-1)^{r+1}}{m}\big\lfloor\frac{m}{lcm(a_{i_1},\ldots,a_{i_r})}\big\rfloor=\frac{-1}{n}+\frac{B(m)}{m}.$$

      So $\frac{B(n)+1}{n}\geq \frac{B(m)}{m}$, and if $B(n)>1$, then $\frac{2B(n)}{n}>\frac{B(n)+1}{n}\geq \frac{B(m)}{m}$.

      When $B(n)=1$, we have $A=\{a\}$ and $a\leq n<2a$. It is clear that $\frac{2B(n)}{n}=\frac{2}{n}>\frac{1}{a}$ and that for any $k\in\mathbb{N}$ and $ka\leq m<(k+1)a$, we have $\frac{B(m)}{m}=\frac{k}{m}\leq \frac{1}{a}$. So $\frac{2B(n)}{n}>\frac{B(m)}{m}$.
