Let $f(n)$ be maximal such that if $B\subset (2n,4n)\cap \mathbb{N}$ there exists some $C\subset (n,2n)\cap \mathbb{N}$ such that $c_1+c_2\not\in B$ for all $c_1\neq c_2\in C$ and $\lvert C\rvert+\lvert B\rvert \geq f(n)$.

Estimate $f(n)$. In particular is it true that $f(n)\leq n^{1/2+o(1)}$?
#788: [Er73]
additive combinatorics

There are no solutions, partial or complete, claimed in the comments.
A conjecture of Choi [Ch71], who proved $f(n) \ll n^{3/4}$. Adenwalla in the comments has provided a simple construction that proves $f(n) \gg n^{1/2}$.

Hunter in the comments has sketched an argument that gives $f(n) \ll n^{2/3+o(1)}$. The bound\[f(n) \ll (n\log n)^{2/3}\]was proved by Baltz, Schoen, and Srivastav [BSS00].

In general, the argument given by Hunter shows that, if a random Cayley graph with probability $p$ has (almost surely) independence number $\ll p^{-c-o(1)}$, then $f(n) \leq n^{\frac{c}{c+1}+o(1)}$.

The work of Alon and Pham [AlPh25] on random Cayley graphs therefore implies that\[f(n) \leq n^{3/5+o(1)},\]and the conjectured bound of $\leq p^{-1+o(1)}$ for the independence number would yield $f(n)\leq n^{1/2+o(1)}$.

  • My colleague Johnny Yu has run this query on GPT 5.2 Pro that reduces this to a graph problem, then applies an existing result from Alon-Pham to produce a tighter upper bound: https://chatgpt.com/share/696e0e87-b834-8012-b0b1-cd9829fe3d21

    They apply the Alon-Pham result (Theorem 4) by constructing a graph $G_B$ of vertices $\{n, ... 2n\}$, then drawing an edge if $v_1 + v_2 \in B$. Then a set $C$ is "admissible" if $C$ is an independent set in $G$, where the best possible size of $C$ is $\alpha(G_B)$. So they try to minimize $\min_B(|B| + \alpha(G_B))$. This graph can be transformed into the same setting that Alon-Pham study, where they show that w.h.p. a full random Cayley sum graph $S$ has independence number at most $O(p^{2/3})$, where $|S|$ is typically $O(np)$. Solving for the optimal $p$ gives $|B|$ + $\alpha(G_B) << n^{3/5}\log(n)^{O(1)}$.

    It doesn't solve whether $f(n) \le n^{1/2 + o(1)}$, but maybe worth updating the body of the problem to include this result.

    • Thanks! In fact this is the same idea as given by Zach Hunter below, but taking advantage of the new independence number bounds by Alon and Pham (which came out after Zach posted his comment). Zach used the previous bound of $\alpha(G) \lesssim p^{-2}$, which balanced against $\ll pn$ yields $p\approx n^{-1/3}$ and hence $n^{2/3+o(1)}$. The better Alon-Pham bound has $\lesssim p^{-3/2}$, whence $p\approx n^{-2/5}$ and hence $n^{3/5+o(1)}$.

      The conjectured bound of $\lesssim n^{1/2+o(1)}$ would follow from the conjectured truth about independence number of Cayley graphs (see conjecture 2 of Alon and Pham).

      So the reduction to the graph theory problem here is not new, but thanks for pointing out the improved bound from the Alon-Pham paper.

  • I think it is likely that if one takes $B$ to be a random set of density $(\log n)^{100}/\sqrt{n}$, then whp $(C+C)\cap B$ is non-empty for all $C$ of size $\sqrt{n}(\log n)^{100}$. indeed, morally, we care about the independence number of the Cayley sum graph generated by $B$ (taking everything modulo $p$ for $p \sim n$). I believe this Cayley graph behaves essentially like a random binomial graph $G$ with the same number of vertices and edges (see: https://arxiv.org/abs/2406.09361 for such a result at a density of $1/\text{polylog}(n)$).

    one can get an improved upper bound of $f(n) \le n^{2/3+o(1)}$, by considering random $B$ of density $\approx n^{-1/3}$. the point is that we get a $n^{2/3+o(1)}$-regular Cayley graph with eigenvalues of size at most $n^{1/3+o(1)}$ (see: Proposition 6' from https://web.math.princeton.edu/~nalon/PDFS/exp1.pdf). therefore by expander mixing lemma, any set of size $n^{2/3+\Omega(1)}$ should contain edges.

  • The lower bound of $n^{1/2}$ is clear. Take any $B=\{b_1<\ldots<b_t\}\subset(2n,4n)\cap\mathbb{N}$ such that $|B|=g(n)$. Then, letting $2n=b_0$ and $4n=b_{t+1}$, we have $\max_{0\leq i\leq t}\{b_{i+1}-b_i\}\geq \frac{2n}{g(n)+1}$. Denote some $i\in\{0,\ldots,t\}$ that satisfies this as $i'$ and choose $C$ to be the interval $(\frac{b_{i'}}{2},\frac{b_{{i'}+1}}{2})\cap\mathbb{N}$. So $C\subset(n,2n)$ and $|B|+|C|\geq g(n)+\frac{n}{g(n)+1}-1\geq 2\sqrt{n}-2$.
