Let $f(z)=\prod_{i=1}^n(z-z_i)\in \mathbb{C}[z]$ with $\lvert z_i\rvert \leq 1$ for all $i$. Let $\rho(f)$ be the radius of the largest disc which is contained in $\{z: \lvert f(z)\rvert< 1\}$.

Determine the behaviour of $\rho(f)$. In particular, is it always true that $\rho(f)\gg 1/n$?
#1039: [EHP58,p.134]
analysis | polynomials

A problem of Erdős, Herzog, and Piranian, who note that $f(z)=z^n-1$ has $\rho(f) \leq \frac{\pi/2}{n}$.

Pommerenke [Po61] proved that\[\rho(f) \geq \frac{1}{2en^2}.\]Krishnapur, Lundberg, and Ramachandran [KLR25] proved\[\rho(f) \gg \frac{1}{n\sqrt{\log n}}.\]

  • Using Claude Opus 4.6 (Anthropic), I obtained a partial result on the Erdős–Herzog–Piranian conjecture ($\rho_n \ge c/n$ for monic degree-$n$ polynomials with roots in the closed unit disc).
    Main result. Call $P$ non-degenerate if every critical value has modulus $\ge 1$. Equivalently, $P$ has no branch points over $\mathbb{D}$ , so every inverse branch $\psi_j$ with $\psi_j(0) = z_j$ extends univalently across all of $\mathbb{D}$, with $\psi_j(\mathbb{D}) \subset \Lambda_P(1)$.

    Theorem. If $P$ is non-degenerate with roots in the closed unit disc, then $\operatorname{inrad}(\Lambda_P(1)) \ge \frac{1}{4n}$.
    Proof sketch. The Koebe quarter theorem gives $\Lambda_P(1) \supset B\left(z_j, \frac{1}{4|P'(z_j)|}\right)$ for each root. The Fekete energy of $n$ points in the closed disc is maximized by $n$-th roots of unity, giving $$\sum_j \log |P'(z_j)| \le n \log n,$$so by pigeonhole $$\min_j |P'(z_j)| \le n.$$Combining: $\operatorname{inrad} \ge \frac{1}{4n}$. The conjectured extremizer $z^n − 1$ is non-degenerate with $|P'(\omega_k)| = n$ for each root.

    Reduction. The full EHP conjecture reduces to: the inradius minimizer is non-degenerate. For degenerate components containing $k$ roots,
    the Riemann map $\varphi : \mathbb{D} \rightarrow \mathcal{C}$ gives a degree-$k$ Blaschke product $B = P \circ \varphi$, and the inradius bound becomes $\operatorname{inrad}(C) ≥ \frac{|B'(0)|}{4|P'(z^*)|}$. For all $k \ge 2$, the hyperbolic Blaschke invariants degenerate as the component approaches a pinch point — for $k = 2$, $$|B''(wc)|(1−|wc|^2)^2 = 2(1−|B(wc)|²) \rightarrow 0$$as the critical value approaches the boundary; for $k \ge 3$, the family $B_a(w) = w^2 \cdot \frac{w−a}{1−\overline{a}w}$ with $a \rightarrow 0$ shows the same degeneration at all critical points simultaneously. This creates a uniform structural obstruction for the conformal approach in the entire degenerate case.

    Numerical evidence. Over several hundred random configurations with $n = 10$ (roots on $\mathbb{T}$, in $\mathbb{D}$, clustered, adversarial), the minimum of $n \cdot \operatorname{inrad}$ was always attained by roots of unity. Degenerate configurations consistently had larger inradii, typically $n \cdot \rho \ge 4$.
    The approach is entirely conformal — no area or perimeter estimates — and is independent of KLR (2025). A literature search (via deep research) found no prior work establishing these results or using these methods.
    Paper: https://github.com/huy209vn/INRADIUS-OF-POLYNOMIAL-LEMNISCATES-THE-NON-DEGENERATE-CASE-AND-BEYOND

    • I have digested this argument and can vouch for its correctness.

      The argument proves that for any polynomial $p(z) = \prod_{i=1}^n |z - z_i|$, $|z_i| \le 1$, with lemniscate $\Omega_p = \{ z: |p(z)| < 1 \}$, there must be a disk centered at some zero $z_i$ with radius at least $(\log 2)/n$ which is contained in $\Omega_p$. This shows $\rho(p) \ge (\log 2)/n \gg 1/n$, answering the second question.

      For a strategy that always produces a disk centered at some zero $z_i$, the $\log 2$ constant is sharp. An example that demonstrates the sharpness is $p(z) = z^n - 1$.

      The idea is a product estimate. If no such disk exists, we can find $w_j$ such that $|p(w_j)| \ge 1$ and $|w_j - z_j| < (\log 2)/n$. This implies
      $$ 1 \le \prod_{j=1}^n |p(w_j)| = \prod_{j=1}^n \prod_{i=1}^n |w_j - z_i|. $$
      However, this product can be upper bounded with standard tools. Specifically, if $0 < c < \log 2$ is such that $|w_j - z_j| < c/n$, then this product is at most $(\exp(c) - 1)^n < 1$, which is a contradiction.

      With GPT-5.5 Thinking, I have also produced a streamlined version of the argument, which can be found in these notes.
