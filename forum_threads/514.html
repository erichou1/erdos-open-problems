OPEN This is open, and cannot be resolved with a finite computation.
Let $f(z)$ be an entire transcendental function. Does there exist a path $L$ so that, for every $n$,\[\lvert f(z)/z^n\rvert \to \infty\]as $z\to \infty$ along $L$?

Can the length of this path be estimated in terms of $M(r)=\max_{\lvert z\rvert=r}\lvert f(z)\rvert$? Does there exist a path along which $\lvert f(z)\rvert$ tends to $\infty$ faster than a fixed function of $M(r)$ (such that $M(r)^\epsilon$)?
#514: [Er61,p.249][Er82e]

  • I would like to share a short note and a Lean 4 formalization related to the third question in Erdős problem #514, both produced using GPT-5.5 Pro (which carried out the natural-language proof and its Lean formalization).

    The note addresses only the third, comparison-function question, in the following universal sense: the comparison function is fixed in advance and is independent of the transcendental entire function $f$. The main result proved is a negative answer in a strong form. For every nondecreasing function $\Phi(T)\to\infty$, there exists a transcendental entire function $f$ such that every path to infinity $\gamma$ satisfies
    $$
    \liminf_{t\to\infty}
    \frac{|f(\gamma(t))|}{\Phi(M_f(|\gamma(t)|))}=0,
    \qquad
    M_f(r)=\max_{|z|=r}|f(z)|.
    $$
    Consequently there is no universal comparison function $\Psi(T)\to\infty$, independent of $f$, for which every transcendental entire function has a path to infinity along which
    $$
    |f(z)|/\Psi(M_f(|z|))\to\infty.
    $$
    In particular, this rules out the parenthetical power-type case $\Psi(T)=T^\varepsilon$.

    The note does not attempt to reprove the first two questions in Problem #514. Its purpose is only to give a self-contained proof of the negative answer to the universal-function version of the third question. The argument is independent of the Langley/Wu-based note previously linked in this thread.

    I would especially welcome corrections concerning references, priority, and the exact interpretation of Erdős’s third question. In particular, if “fixed function of $M(r)$” has a standard interpretation in the literature different from the universal-comparison-function interpretation used here, I would be grateful to know.

    • I have digested the proof for the first two questions and can vouch for its correctness.

      Essentially, the result is a short derivation from this 1985 paper by Wu. Wu's Theorem B concerns growth of a subharmonic function along paths. We apply the theorem to the subharmonic function $u = \max(\log |f|, 0)$. For a transcendental entire $f$, one has $\log M(r, f)/\log r \to \infty$, which gives the desired hypothesis to use the theorem. Wu's theorem then gives both existence of the required path and its length estimate.
