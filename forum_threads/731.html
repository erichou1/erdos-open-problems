Find some reasonable function $f(n)$ such that, for almost all integers $n$, the least integer $m$ such that $m\nmid \binom{2n}{n}$ satisfies\[m\sim f(n).\]
#731: [EGRS75]
number theory | binomial coefficients

  • Using Chatgpt 5.4 i get an heuristic which seems to suggest a possible refinement of the estimate $m=\exp((\log n)^{1/2+o(1)})$, but also suggests that the asymptotic $m\sim f(n)$ may not exist in the usual deterministic sense.

    Let $a(n)=\min{m\geq 1:m\nmid {2n\choose n}}$. Since $a(n)=\min_p p^{v_p({2n\choose n})+1}$, the smallest missing prime already gives a natural candidate for the scale of $a(n)$.

    By Kummer's theorem, for odd $p$, $p\nmid {2n\choose n}$ if and only if every base-$p$ digit of $n$ lies in ${0,1,\ldots,(p-1)/2}$. Thus, for a random $n\leq x$, writing $L=\log x$, one expects ${\bf P}(p\nmid {2n\choose n})\approx 2^{-L/\log p}=\exp(-(\log 2)L/\log p)$.

    Therefore the expected number of primes $p\leq y=e^u$ which do not divide ${2n\choose n}$ should be roughly

    $\sum_{p\leq e^u}\exp(-(\log 2)L/\log p)\sim {\exp(u-(\log 2)L/u)\over u(1+(\log 2)L/u^2)}.$

    Equating this to $1$ gives

    $u=\sqrt{(\log 2)L}+{1\over 4}\log L+{1\over 4}\log(\log 2)+{1\over 2}\log 2+o(1).$

    Thus the natural scale is

    $F(x)=\sqrt2,(\log 2)^{1/4}(\log x)^{1/4}\exp(\sqrt{(\log 2)(\log x)}).$

    More generally, if $y=rF(x)$, then the same calculation gives expected number asymptotic to $r^2$. Assuming a Poisson-type independence for these digit-restriction events as $p$ varies, this suggests

    $ {1\over x}\,\#\{n\leq x:\ a(n)>rF(x)\}\to e^{-r^2} $

    for each fixed $r>0$. Equivalently,

    $\log a(n)=\sqrt{(\log 2)(\log n)}+{1\over 4}\log\log n+O(1)$

    in distribution.

    This would refine the EGRS estimate, but it would also suggest that there is no function $f(n)$ with $a(n)\sim f(n)$ for almost all $n$, unless the Poisson heuristic is misleading. The missing point is to prove, or disprove, the required independence statement for the simultaneous base-$p$ digit restrictions in the range

    $p\asymp(\log x)^{1/4}\exp(\sqrt{(\log 2)(\log x)}).$

    Prime powers with $v_p({2n\choose n})>0$ appear heuristically not to change this leading distribution, since the relevant low-carry events for $p<y$ should have much smaller total contribution.
