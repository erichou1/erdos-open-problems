Let $S\subset \mathbb{R}$ be a set containing no solutions to $a+b=c$. Must there be a set $A\subseteq \mathbb{R}\backslash S$ of cardinality continuum such that $A+A\subseteq \mathbb{R}\backslash S$?
#949: [Er77c]
ramsey theory

There are no solutions, partial or complete, claimed in the comments.
Erdős suggests that if the answer is no one could consider the variant where we assume that $S$ is Sidon (i.e. all $a+b$ with $a,b\in S$ are distinct, aside from the trivial coincidences).

In the comments Dillies gives a positive proof of this, found by AlphaProof: in other words, if $S\subset \mathbb{R}$ is a Sidon set then there exists $A\subseteq \mathbb{R}\backslash S$ of cardinality continuum such that $A+A\subseteq \mathbb{R}\backslash S$.

  • I can prove the problem when $S$ has Baire property (a bit orthogonal to $S$ being Sidon), and also remark that Yael Dillies argument give a proof for $|S|< \mathfrak{c}$ without assuming $S$ being Sidon. So if there's a counter example in general to it, it would be very pathological (non Sidon, non measurable, non BP).

    So let's assume $S$ has BP. We split into cases.

    Case 1: $\mathbb R\setminus S$ contains a neighborhood of $0$.
    Then there is $\varepsilon>0$ such that $(-\varepsilon,\varepsilon)\cap S=\varnothing$.
    Let $A=(0,\varepsilon/2)$. Then $A\subseteq\mathbb R\setminus S$ and\[
    A+A=(0,\varepsilon)\subseteq\mathbb R\setminus S,
    \]so we are done.

    Case 2: $\mathbb R\setminus S$ contains no neighborhood of $0$.
    Then $0\in\overline S$.

    Claim: $S$ is meagre.
    Suppose for a contradiction that $S$ is not meagre. Since $S$ has the Baire property,
    there exists a nonempty open interval $I=(a,b)$ such that $C:=S\cap I$ is comeagre in $I$.
    Let $\ell:=b-a>0$. As $0\in\overline S$, choose $s\in S$ with $0<|s|<\ell$.
    Then the overlap\[
    J:=I\cap(I-s)
    \]is a nonempty open interval. Since $C$ is comeagre in $I$, the set $C\cap J$ is comeagre in $J$.
    Also $C-s$ is comeagre in $I-s$, hence $(C-s)\cap J$ is comeagre in $J$.
    Therefore\[
    D:=(C\cap J)\cap\bigl((C-s)\cap J\bigr)
    \]is comeagre in $J$, in particular $D\neq\varnothing$. Pick $x\in D$. Then $x\in C\subseteq S$,
    and $x\in C-s$ implies $x+s\in C\subseteq S$. Together with $s\in S$ we get
    $x+s\in S\cap(S+S)$, contradicting that $S$ is sum-free. Hence $S$ is meagre.


    Now define the set of ''bad pairs'' $R\subseteq\mathbb R^2$ by\[
    R \;:=\; (S\times\mathbb R)\ \cup\ (\mathbb R\times S)\ \cup\ \{(x,y)\in\mathbb R^2:\ x+y\in S\}.
    \]Since $S$ is meagre in $\mathbb R$, both $S\times\mathbb R$ and $\mathbb R\times S$
    are meagre in $\mathbb R^2$.
    For the third set, write $S=\bigcup_{n} N_n$ where each $N_n$ is nowhere dense.
    Let $\phi:\mathbb R^2\to\mathbb R$ be $\phi(x,y)=x+y$. The map $\phi$ is continuous and open,
    so for each $n$ the preimage $\phi^{-1}(N_n)$ is nowhere dense in $\mathbb R^2$:
    indeed, if $U\subseteq\mathbb R^2$ is nonempty open, then $\phi(U)$ is nonempty open in $\mathbb R$,
    so $\phi(U)\not\subseteq \overline{N_n}$, hence $U\not\subseteq \overline{\phi^{-1}(N_n)}$.
    Thus $\phi^{-1}(S)=\bigcup_n \phi^{-1}(N_n)$ is meagre.
    Consequently $R$ is meagre in $\mathbb R^2$.

    By Mycielski's theorem, there exists a perfect set $P\subseteq\mathbb R$ such that\[
    (P\times P)\cap R=\varnothing.
    \]Then for all $x,y\in P$ we have $x\notin S$, $y\notin S$, and $x+y\notin S$.
    Hence $P\subseteq\mathbb R\setminus S$ and $P+P\subseteq\mathbb R\setminus S$.
    Taking $A:=P$ completes the proof.

  • For the question from the additional material:
    “Erdős suggests that if the answer is no, one could consider the variant where we assume that $S$ is Sidon.”
    AlphaProof found the following solution:

    We case on whether $S$ has cardinality the continuum or strictly less.
    If $S$ has cardinality strictly less than the continuum, then we pick by Zorn some maximal $A$ such that both $A$ and $A + A$ are disjoint from $S$. Now, to see $A$ has size the continuum, note that
    $$A \cup \bigcup_{a \in A} (S - a) \cup S \cup S/2 = \mathbb R$$
    by maximality of $A$. By assumption, $|S^c \cap (S / 2)^c| = |\mathbb R|$. If $|A| < |\mathbb R|$, we would therefore have
    $$|\mathbb R| = |S^c \cap (S / 2)^c| ≤ |A \cup \bigcup_{a \in A} (S - a)| ≤ |A| + |A| |S| <|\mathbb R|,$$
    contradiction.
    If $S$ has cardinality the continuum, then we pick some $a\ne 0$ in $S$ and set $A := (S \setminus \{a\} - a / 2) \setminus S$. Since $S$ is Sidon and $a\ne 0$,
    $$(S - a / 2) \cap S \supseteq (S \setminus \{a\} - a / 2) \cap S$$
    has at most one element. In particular, $|A| = |S \setminus \{a\} - a / 2| = |S| = |\mathbb R|$ as wanted.
    Since $S$ is Sidon,
    $$A + A \subseteq S \setminus \{a\} + S \setminus \{a\} - a$$
    is disjoint from $S$, as wanted.

  • The answer is no. Let $S$ be the set of all intervals of the form $[3n+1, 3n + 2)$, with $n \in \mathbb{Z}$. Then $S$ is sum-free. Furthermore, if $a \in \mathbb{R}\backslash S$ and $a$ is not a multiple of $3$, then $a$ has a multiple in $S$. As such, the only subsets of $\mathbb{R}\backslash S$ that are closed under addition are the sub-semigroups of $3\mathbb{Z}$, so $\mathbb{R}\backslash S$ certainly has no uncountable subsets that are closed under addition.


    The website mentions the variant where we instead require that $S$ is Sidon. That's probably a lot harder, and my construction is very much not Sidon.
