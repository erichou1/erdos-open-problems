PROVED (LEAN) This has been solved in the affirmative and the proof verified in Lean.
There exists a constant $C>0$ such that, for all large $N$, if $A\subseteq \{1,\ldots,N\}$ has size at least $\frac{5}{8}N+C$ then there are distinct $a,b,c\in A$ such that $a+b,a+c,b+c\in A$.
#865: [Er72][CES75][Er92c]
number theory | additive combinatorics
A problem of Erdős and Sós (also earlier considered by Choi, Erdős, and Szemerédi [CES75], but Erdős had forgotten this). Taking all integers in $[N/8,N/4]$ and $[N/2,N]$ shows that $\frac{5}{8}$ would be best possible here.

It is a classical folklore fact that if $A\subseteq \{1,\ldots,2N\}$ has size $\geq N+2$ then there are distinct $a,b\in A$ such that $a+b\in A$, which establishes the $k=2$ case.

In general, one can define $f_k(N)$ to be minimal such that if $A\subseteq \{1,\ldots,N\}$ has size at least $f_k(N)$ then there are $k$ distinct $a_i\in A$ such that all $\binom{k}{2}$ pairwise sums are elements of $A$. Erdős and Sós conjectured that\[f_k(N)\sim \frac{1}{2}\left(1+\sum_{1\leq r\leq k-2}\frac{1}{4^r}\right) N,\]and a similar example shows that this would be best possible.

Choi, Erdős, and Szemerédi [CES75] have proved that, for all $k\geq 3$, there exists $\epsilon_k>0$ such that (for large enough $N$)\[f_k(N)\leq \left(\frac{2}{3}-\epsilon_k\right)N.\]This problem was solved in the affirmative by Cipollini and GPT Pro [Ci26].

  • I believe I have a candidate proof of the $k=3$ case (which would resolve the problem), giving the sharp bound $f_3(N)\leq \frac{5}{8}N+O(1)$. The main new ingredient is a folded additive lemma. Roughly, if $B\subseteq \{1,\ldots,m-1\}$ is cyclically weakly sum-free modulo $m$, and if $C$ denotes the set of residues which occur both as a non-wrapped and wrapped pair-sum of two distinct elements of $B$, then $|B|-|C|\leq \frac{m}{4}+O(1)$.

    Folding $A$ around a central element converts these collisions into empty folded coordinates. This gives the $5/8$ bound after a centering argument; the off-center case uses the Choi, Erdős, and Szemerédi $<2/3$ theorem as a coarse input.

        Finally, using lemma 3.1 the main result is concluded in section 4; here $2$ cases are considered depending on the choice of $h$ being slightly larger or slightly smaller than $N/2.$

  • If it is true for large N, it will be for all N (possibly with a slightly larger constant C).

    Maybe interestingly, for $N=8i+6$, where $i \le 6$, the maximum is $5i+5$, except for $N=14$ when the maximum is $11.$
    The latter attained e.g. by the set $\{1,2,4\} \cup [N/2, N]$.

    For $N=8i$ and $3\le i \le 7$, the construction $[N/8,N/4]\cup[N/2,N]$ is actually the only extremal construction (not containing a subset of the form $\{a,b,c,a+b,b+c,a+c\}$) attaining size $5/8N+2$.

    So this suggests that likely the answer to the question is yes (and $C$ can be even taken rather small).
