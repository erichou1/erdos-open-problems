Let $c>0$. If $x$ is sufficiently large then does there exist $n\leq x$ such that the values of $\phi(n+k)$ are all distinct for $1\leq k\leq (\log x)^c$, where $\phi$ is the Euler totient function?

Erdős, Pomerance, and Sárközy [EPS87] proved that if $\phi(n+k)$ are all distinct for $1\leq k\leq K$ then\[K \leq \frac{n}{\exp(c(\log n)^{1/3})}\]for some constant $c>0$.

See [945] for the analogous problem with the divisor function.

  • Gpt 5.5 pro found a partial result


    Let \(L=L(x)\) satisfy\[
    L\log(2L)=o((\log x)^2).
    \]Then for all but \(o(x)\) integers \(n\le x\), the values\[
    \varphi(n+1),\varphi(n+2),\ldots,\varphi(n+L)
    \]are pairwise distinct. In particular, this gives an affirmative answer to the problem for every fixed exponent \(c<2\), taking \(L=\lfloor(\log x)^c\rfloor\). It also gives, for example,\[
    L=\frac{(\log x)^2}{(\log\log x)^{1+\varepsilon}}
    \]for any fixed \(\varepsilon>0\).

    The broad idea is to count bad starting points. If \(n\) is bad, then for some \(1\le i<j\le L\),\[
    \varphi(n+i)=\varphi(n+j).
    \]Writing \(h=j-i\) and \(m=n+i\), this gives a shifted equal-totient equation\[
    \varphi(m)=\varphi(m+h).
    \]Thus the number of bad starts is bounded by\[
    \sum_{h<L}(L-h)P(x+L;h),
    \]where\[
    P(X;h)=\#\{m\le X:\varphi(m)=\varphi(m+h)\}.
    \]The note uses the Graham--Holt--Pomerance / Pollack--Pomerance--Treviño decomposition (references in note)\[
    P(X;h)=P_0(X;h)+P_1(X;h).
    \]The exceptional part \(P_1\) is exponentially small uniformly for polylogarithmic \(h\). The structured part satisfies\[
    P_0(X;h)\ll c(h)\frac{X}{(\log X)^2}.
    \]The extra observation in the note is that the structured constants have enough average saving:\[
    \sum_{h\le L} c(h)\ll \log L.
    \]Combining these gives\[
    \begin{aligned}
    \#\{\text{bad }n\le x\}
    &\ll
    \frac{x}{(\log x)^2}
    \sum_{h<L}(L-h)c(h)
    +
    o(x) \\
    &\ll
    x\frac{L\log L}{(\log x)^2}+o(x).
    \end{aligned}
    \]This is \(o(x)\) whenever\[
    L\log L=o((\log x)^2).
    \]So this does not solve the full problem for arbitrary fixed \(c>0\), but it does give a rigorous almost-all result up to essentially the \((\log x)^2\) scale.

  • A slightly stronger statement follows from the usual shifted-collision estimates for the totient function. Let \(c>0\) be fixed and put \(L=\lfloor(\log x)^c\rfloor\). I will use the following standard consequence of the Erdős--Pomerance--Sárközy/Graham--Holt--Pomerance primitive-set method: for every \(A,B>0\), uniformly for \(1\le h\le(\log X)^B\), one has\[
    R_h(X):=\#\{m\le X:\varphi(m)=\varphi(m+h)\}\ll_{A,B}\frac{X}{(\log X)^A}.
    \]This is unconditional. Call \(n\le x-L\) bad if the values \(\varphi(n+1),\ldots,\varphi(n+L)\) are not all distinct. If \(n\) is bad, then there exist \(1\le i<j\le L\) such that \(\varphi(n+i)=\varphi(n+j)\). Writing \(h=j-i\) and \(m=n+i\), this gives \(\varphi(m)=\varphi(m+h)\), with \(1\le h\le L\) and \(m\le x+L\). For fixed \(h\), there are at most \(L-h\le L\) possible values of \(i\). Hence the number of bad \(n\le x-L\) is at most\[
    \sum_{h=1}^{L-1}(L-h)R_h(x+L)\le L\sum_{h=1}^{L}R_h(x+L).
    \]Taking \(B>c\) in the shifted-collision estimate gives, for every \(A>0\),\[
    R_h(x+L)\ll_{A,c}\frac{x}{(\log x)^A}
    \]uniformly for \(1\le h\le L\). Therefore\[
    \#\{n\le x-L:n\text{ is bad}\}\ll_{A,c}L^2\frac{x}{(\log x)^A}\ll_{A,c}x(\log x)^{2c-A}.
    \]Choosing \(A>2c+1\), this is \(o(x)\). Consequently almost every \(n\le x\) has the property that \(\varphi(n+1),\ldots,\varphi(n+\lfloor(\log x)^c\rfloor)\) are pairwise distinct. In particular, for all sufficiently large \(x\), there exists some \(n\le x\) such that the values \(\varphi(n+k)\), \(1\le k\le(\log x)^c\), are all distinct.

      • Yes Chojecki's objection is correct: the displayed estimate\[
        R_h(X)\ll_{A,B} X(\log X)^{-A}
        \]uniformly for \(1\le h\le(\log X)^B\) does not seem to be a standard consequence
        of the Erdős--Pomerance--Sárközy/Graham--Holt--Pomerance method. What is known
        is more like a decomposition into the structured part \(P_0(x;h)\), of size
        \(\ll c(h)x/(\log x)^2\), and an exceptional part \(P_1(x;h)\), for which one
        does get much stronger exponential-log savings uniformly in \(h\). The
        structured part is precisely the obstruction to obtaining arbitrary powers of
        \(\log x\).

        Thus the union-bound argument after the estimate is fine, but the quoted
        estimate itself is too strong as stated. With the known \(x/(\log x)^2\)-type
        uniform bound one only gets a much more limited polylogarithmic range, not the
        claimed statement for every fixed \(c>0\).
