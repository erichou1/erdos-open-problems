OPEN This is open, and cannot be resolved with a finite computation.
If $n=\prod_{1\leq i\leq t} p_i^{k_i}$ is the factorisation of $n$ into distinct primes then let\[f(n)=\sum p_i^{\ell_i},\]where $\ell_i$ is chosen such that $n\in [p_i^{\ell_i},p_i^{\ell_i+1})$. Furthermore, let\[F(n)=\max \sum_{i} a_i\]where the maximum is taken over all distinct $a_1,\ldots,a_k\leq n$ such that $(a_i,a_j)=1$ for $i\neq j$ and all prime factors of each $a_i$ are prime factors of $n$.

Is it true that, for almost all $n$,\[f(n)=o(n\log\log n)\]and\[F(n) \gg n\log\log n?\]Is it true that\[\max_{n\leq x}f(n)\sim \frac{x\log x}{\log\log x}?\]Is it true that (for all $x$, or perhaps just for all large $x$)\[\max_{n\leq x}f(n)=\max_{n\leq x}F(n)?\]Find an asymptotic formula for the number of $n<x$ such that $f(n)=F(n)$. Find an asymptotic formula for\[H(x)=\sum_{n<x}\frac{f(n)}{n}.\]Is it true that\[H(x) \ll x\log\log\log\log x?\]
#878: [Er84e][Er98,p.172]

There are no solutions, partial or complete, claimed in the comments.
Erdős [Er84e] proved that\[\max_{n\leq x}f(n)\sim \frac{x\log x}{\log\log x}\]for a sequence of $x\to \infty$.

It is trivial that $f(n)\leq F(n)$ for all $n$. It may be true that, for almost all $n$,\[F(n)\sim \frac{1}{2}n\log\log n.\]Erdős notes that $f(n)/n$ 'almost behaves as a conventional additive function', but unusually $f(n)/n$ does not have a mean value - indeed,\[\limsup \frac{1}{x}\sum_{n<x}\frac{f(n)}{n}=\infty\]but\[\liminf \frac{1}{x}\sum_{n<x}\frac{f(n)}{n}<\infty.\]Erdős [Er84e] proved that\[x\log\log\log\log x\ll H(x) \ll x\log\log\log x.\]Barreto has noted in the comments that this upper bound for $H(x)$ implies $f(n)=o(n\log\log n)$ for almost all $n$. Barreto has also noted that $\max_{n\leq x}f(n)=\max_{n\leq x}F(n)$ is false at $x=210$.

See also [879].

  • A computer search disproves $\max_{n\leq x}f(n)=\max_{n\leq x}F(n)$ being true for all $x$. Take $x=2\cdot 3\cdot 5\cdot 7=210$, then $$f(210)=2^7+3^4+5^3+7^2=128+81+125+49=383.$$ One can easily verify that indeed no value of $n\leq 210$ gives a larger value for $f(n)$. But now consider the admissible choice $$(a_1,a_2,a_3,a_4)=(2^7,3^3\cdot 7,5^3,1)=(128,189,125,1)$$ which are pairwise coprime, each at most $210$, and all of whose prime factors lie in the set of prime divisors of $210$, namely $\{2,3,5,7\}$. Then, $$F(210)\geq 128+189+125+1=443,$$ so $$\max_{n\leq 210}F(n)\geq 443>383=\max_{n\leq 210}f(n).$$

      • Hah, I was about to edit my comment to remark on me realising that this assumes 1 is valid! I agree, this is a bit of a dodgy case.

        EDIT: Ah, wait, I believe we do have equality of maxima (in fact, equality of $F_{>1}(n)$ and $f(n)$!) when we force the $a_i$ to all be strictly greater than $1$!

        Let $P(n):=\{p\in\mathbb{P}:p\mid n\}=\{p_1,\,\dots,\,p_t\}$ be the set of prime divisors of $n$ and define $\ell_p:=\max\{l:p^l\leq n\}$, $f(n)=\sum_{p\in P(n)}p^{\ell_p}$ as in the problem. Take any admissible tuple $(a_1,\,\dots,\,a_t)$ with all $a_i>1$ and let $S_i:=\{p\in P(n):p\mid a_i\}$. Each $S_i$ is non-empty, and since the $a_i$ are pairwise coprime, the $S_i$ are pairwise disjoint, hence $$t\leq\sum_{i=1}^{t}\left|S_i\right|\leq\left|P(n)\right|=t.$$ Therefore, each $S_i$ has size $1$, so after reordering, $a_i=p_i^{e_i}$ for some $e_i\geq 1$. Since $a_i\leq n$, we have $e_i\leq\ell_{p_i}$, so $$\sum_{i=1}^{t}a_i\leq\sum_{i=1}^{t}p_i^{\ell_{p_i}}=f(n).$$ Thus, $F_{>1}(n)\leq f(n)$. The reverse inequality is indeed immediate since $(p_1^{\ell_{p_1}},\,\dots,\,p_t^{\ell_{p_t}})$ is an admissible tuple with all entries $>1$, and its sum is $f(n)$. So in fact, $F_{>1}(n)=f(n)$ meaning any counterexample to $\max_{n\leq x}F(n)=\max_{n\leq x}f(n)$ must have at least one $a_i=1$. In the original problem statement, there are exactly $t=\omega(n)$ of the $a_i$, so it doesn't seem likely to me that $a_i=0$ can be taken as a value (even ignoring it from the coprimality condition) in the intended sense of the problem.

        So, TL;DR, it is trivially true if $a_i>1$, not true for all $x$ if $a_i=1$ is allowed, or something is off about the problem statement itself if there are not to be exactly $t$ $a_i$, or if $0$ is to be allowed as a value, ignoring it from the coprimality condition.

        • Also, FWIW, something else to note: From $H(x)\ll x\log\log\log x$, one can show $f(n)=o(n\log\log n)$ for almost all $n$.
          Fix $\varepsilon>0$ and let $E_\varepsilon(x):=\{n<x:f(n)>\varepsilon n\log\log n\}$. The integers $n<\sqrt{x}$ contribute at most $\sqrt{x}=o(x)$, so it suffices to count bad $n\in[\sqrt{x},x)$. For such $n$, $$\log\log n\geq\log\log\sqrt{x}=\log\log x-\log 2=(1+o(1))\log\log x$$ uniformly as $x\to\infty$. Hence, for all sufficiently large $x$, $$\log\log n\geq\frac{1}{2}\log\log x\qquad(\sqrt{x}\leq n<x).$$ Therefore, every $n\in E_\varepsilon(x)\cap[\sqrt{x},x)$ contributes $$\frac{f(n)}{n}>\varepsilon\log\log n\geq\frac{\varepsilon}{2}\log\log x$$ to $H(x)$. Thus, $$\frac{\varepsilon}{2}\log\log x\cdot\left|E_\varepsilon(x)\cap[\sqrt{x},x)\right|\leq\sum_{n<x}\frac{f(n)}{n}=:H(x)\ll x\log\log\log x.$$ So, $$\left|E_\varepsilon(x)\cap[\sqrt{x},x)\right|\ll_\varepsilon x\frac{\log\log\log x}{\log\log x}=o(x).$$ Adding back the $O(\sqrt{x})$ discarded integers gives $\left|E_\varepsilon(x)\right|=o(x)$. Since this holds for every fixed $\varepsilon>0$, we obtain $f(n)=o(n\log\log n)$ for almost all $n$.
