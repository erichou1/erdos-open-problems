For any $k\geq 2$ let $g_k(n)$ denote the maximum value of\[(a_1+\cdots+a_k)-n\]where $a_1,\ldots,a_k$ are integers such that $a_1!\cdots a_k! \mid n!$. Can one show that\[\sum_{n\leq x}g_k(n) \sim c_k x\log x\]for some constant $c_k$? Is it true that there is a constant $c_k$ such that for almost all $n<x$ we have\[g_k(n)=c_k\log x+o(\log x)?\]
#400: [ErGr80,p.77]
number theory | factorials

Erdős and Graham write that it is easy to show that $g_k(n) \ll_k \log n$ always, but the best possible constant is unknown.

See also [401].

  • Using a recent result of Pomerance, one gets the lower bound\[
    \sum_{n\le x} g_k(n) \gg_k x\log x
    \]for every fixed \(k\ge 2\).

    Indeed, by Theorem 1 here: https://math.colgate.edu/~integers/aa47/aa47.pdf, for every fixed \(\eta<1/\log 4\), there is a set \(S_\eta\) of asymptotic density \(1\) such that, for every \(m\in S_\eta\) and every positive integer \(r\le \eta\log m\),\[
    \frac{(m+r)!}{m!}\mid {2m\choose m}.
    \]Taking \(r=\lfloor \eta\log m\rfloor\), this gives\[
    m!(m+r)! \mid (2m)!.
    \]Hence, in the definition of \(g_k(2m)\), we may take\[
    a_1=m+r,\qquad a_2=m,\qquad a_3=\cdots=a_k=1,
    \]and obtain\[
    g_k(2m)\ge r+k-2\ge \lfloor\eta\log m\rfloor .
    \]Therefore, with \(M=\lfloor x/2\rfloor\),\[
    \sum_{n\le x}g_k(n)
    \ge
    \sum_{\substack{m\le M\\ m\in S_\eta}}\lfloor\eta\log m\rfloor
    =
    \left(\frac{\eta}{2}+o(1)\right)x\log x.
    \]Since this holds for every \(\eta<1/\log4\),\[
    \liminf_{x\to\infty}\frac{1}{x\log x}\sum_{n\le x}g_k(n)
    \ge \frac{1}{2\log4}.
    \]Together with the standard upper bound \(g_k(n)\ll_k\log n\), this gives the correct order of magnitude for the summatory function, although not the asymptotic constant.

    With some optimization, I think I can get $c_k \ge \frac{k - 1}{\log{4}}$, but the correct constant is likely $\frac{k - 1}{\log{2}}$ as mentioned in the upper bound comment.

  • FWIW, here's what Erdős and Graham probably had in mind: The condition is equivalent to $\sum_i \nu_p(a_i!)\le \nu_p(n!)$ for all $p$. This is equivalent to $\sum_i a_i-s_p(a_i)\le n-s_p(n)$, or $(a_1+\dots+a_k)-n\le s_p(a_1)+\dots+s_p(a_k)-s_p(n)$. If you take $p=2$ for example, you get that $g_k(n)\le \log_2(a_1)+\dots+\log_2(a_k)+O_k(1)\le k\log n+O_k(1)$. This yields the $g_k(n)\ll_k \log n$.

    This idea in fact gives the upper bound $c_k\le \frac{k-1/2}{\log 2}$. Getting a lower bound seems difficult in comparison. Really, the bottleneck should be $p=2$.
