Let $\epsilon>0$. Does there exist $A\subseteq \mathbb{N}$ such that the lower density of $A+A$ is at least $1-\epsilon$ and yet $1_A\ast 1_A(n) \ll_\epsilon 1$ for all $n$?
#749: [Er94b,p.263]
additive combinatorics

It is unclear from [Er94b] whether Erdős believed the answer to this should be yes or no. He also asked a similar question for upper density, for which he seemed more certain the answer should be yes.

He wrote that both he and Sárközy believed that if $1_A\ast 1-A(n)\leq C$ for all $n$ then the upper density of $A+A$ is at most $1-\epsilon_C$, where $\epsilon_C>0$ may depend on $C$.

The upper density variant has been resolved by Bhalla, with the assistance of GPT 5.4. Bhalla constructs, for any $\epsilon>0$, a set $A$ for which the upper density of $A+A$ is at least $1-\epsilon$ and yet $1_A\ast 1_A(n)\ll \epsilon^{-1}$ for all $n$.

See also [28].

    • Nice! Here is my interpretation of the argument. The main task is to solve the single scale problem: for a sequence of rapidly growing scales $M_j$, find finite sets $P_j$ in $[1,M_j]$ such that $P_j + P_j$ covers most of $[1,M_j]$ with bounded multiplicity. The final set $A$ will be the union of the $P_j$; this gives large upper density to $A+A$ but doesn't control lower density. To avoid the multiplicity from blowing up when combining the $P_j$ together, one also needs each $P_j$ to be somewhat well-separated; basically one needs each $P_j$ to have only bounded intersection with intervals of length $M_{j-1}$.

      One takes $M_j$ to be a moderately large multiple of $p_j^2$ for a large prime $p_j$. Using a standard base $p_j$ type representation (actually it's more like base $2p_j$ to deal with some technical digit carry issues), one can reduce matters to a finite field analogue: find a subset $B$ of the finite plane ${\bf F}_{p_j}^2$ such that $B+B$ covers most of the plane with bounded multiplicity, and such that every horizontal line in the plane has bounded intersection with $B$. But this can be accomplished by taking $B$ to be the union of a bounded number of parabolas (this is a fairly standard trick in Sidon set type constructions; the argument here cites inspiration from a construction of Liang, Zhang, and Zuo).

      In this construction, each scale $M_j$ behaves like the square of the previous scale $M_{j-1}$ (to avoid blowup in the multiplicity), so one only has quite a sparse set of scales for which one has good control on the density of the sumset. It seems the lower density problem remains quite challenging; I don't see an easy way to modify this construction to get good sumset density control at all scales (rather than a very sparse set of scales) while simultaneously keeping the multiplicity bounded when combining contributions from multiple scales. On the other hand, disproving the lower density version would also prove #28, which is a notorious open problem.

      EDIT: A toy problem for the lower density version would be to solve a "two scale" problem. Suppose one has two scales $M_1 < M_2$ that are only slightly separated from each other, e.g., $M_1 = M_2 / \log \log M_2$. The challenge is then to find finite sets $P_1 \subset [1,M_1]$, $P_2 \subset [1,M_2]$ such that

      1. $P_1 + P_1$ has bounded multiplicity and occupies most of $[1,M_1]$;
      2. $P_2 + P_2$ has bounded multiplicity and occupies most of $[1,M_2]$;
      3. $P_1 + P_2$ also has bounded multiplicity.
      One can use separate finite field constructions, using separate primes $p_1,p_2$, to achieve Goal 1 and Goal 2... but the problem is Goal 3. All known constructions would require the primes $p_1,p_2$ to be distinct from each other, and then there is no known way to "coordinate" the constructions to also achieve Goal 3 without assuming very wide separation between scales (e.g., $M_2 \gg M_1^2$). (Randomization can help - for instance if $P_1 + P_1$ only covers a positive density fragment of $[1,M_1]$, one can beef it up to cover a lot more of $[1,M_1]$ by adding in various random translates of $P_1$ - but trying to use random constructions to ensure bounded multiplicity of $P_1+P_2$ runs into the standard problem of picking up some logarithmic losses due to tail events in the concentration of measure bounds. Subrandom constructions are what are needed here.) In principle, if one can solve this toy problem (and various generalisations thereof) then there is a path to gluing everything together to get large lower density (or at least large density on a reasonably dense set of scales), but I doubt one can get a full solution to this problem without first addressing this toy problem.

            AronBhalla — 21:04 on 06 Apr 2026

          • I think I agree with this assessment. Using the heuristic that a long interval $[1,MN]$ behaves (for this sort of additive problem) like a Cartesian product of $[1,M]$ and $[1,N]$, one can view two similar scales $[1,M_1]$, $[1,M_2]$ as the Cartesian product of two widely separated scales $[1,L_1]$, $[1,L_2]$ with some common scale $[1,N]$, and one can then take a "Cartesian product" of constructions that work for the two component scales to get something that works for the combined scales. There are some technical issues due to "digit carrying" or "wraparound" ($[1,MN]$ is not *exactly* Freiman-isomorphic to $[1,M] \times [1,N]$) but this only causes some factor of 2 type losses in the constants, which is acceptable.

            So this does technically solve the two-scale problem, but not in a way that allows for iteration, as there is still a large prime $p$ that is common to the construction at both scales. One still needs a way to gracefully transition from one prime to another as one keeps going up to larger scales without causing a large amount of collisions at the transition point. This is hard to model by something concrete like the two-scale problem though. Informally speaking one needs a construction where $P_1$ and $P_2$ are allowed to be fundamentally "different" at small scales, but this is hard to quantify rigorously. On the other hand the above constructions suggest that such transitions only have to occur relatively rarely - e.g., one such transition between one hyperdyadic scale $2^{2^j}$ and the next $2^{2^{j+1}}$. Outside of such transitions one may be able to re-use blocks and get a lot of other scales covered "for free" without making the number of collisions noticeably worse.

            EDIT: One could try to re-use the same blocks (and same primes) indefinitely for arbitrarily large scales, but this would mean that eventually the construction would involve multi-level constructions where the number $k$ of levels is not something fixed like $1$ or $2$, but goes to infinity as the scale increases. Because of all the issues with digit carrying, I think all the constants here are going to grow exponentially in $k$, so this is not going to be sustainable indefinitely. The requirement that $1_A * 1_A$ stay bounded becomes quite challenging to satisfy when $A$ starts consisting of multiple components from multiple scales!

            SECOND EDIT: Here is perhaps a proposed second toy problem. Instead of having two scales $M_1 < M_2$, have $K$ scales $M_1 < \dots < M_K$ where $M_k = M_1 (\log\log M_1)^k$ say, and $K$ is a medium-sized constant (and $M_1$ sufficiently large depending on $K$). The challenge is now to find $P_k \in [1,M_k]$ where $P_k + P_k$ occupies a decent fraction of $[1,M_k]$ (e.g., $\gg 1$ of this interval) with not just $1_{P_k} * 1_{P_k}$ bounded, but $1_{P_k} * 1_{P_{k'}} \ll 1$ for all $1 \leq k < k' \leq K$. But here is the new twist: the implied constants need to be independent of $K$. I could imagine iterating the previous construction to give something like this, but for which all the bounds grow exponentially in $K$. But to get to the full upper density result I think one will need uniformity in $K$.

            One final note: if there is a power gap between scales, say $M_1 \leq M_2^{0.99}$, then standard probabilistic heuristics suggest that $1_{P_1} * 1_{P_2}$ will almost certainly be bounded. (One may need to replace the parabolas in the finite field construction by higher degree curves to get enough randomization for the probabilistic argument to be rigorous). So it is mostly about trying to understand the interaction of very close scales.

    • Great! This construction is similar to that in Ruzsa's paper (which I think is the origin for this kind of construction).

      Indeed, Ruzsa constructs, for infinitely many $N$ (namely those of the shape $3p^2$ where $(\frac{2}{p})=-1$), a set $A\subset [1,N]$ such that $A+A$ contains the interval $[\frac{2}{3}N,\frac{4}{3}N]$, and $1_A\ast 1_A(n)\leq 288$ for all $n$. (These blocks have some further nice properties, like small intersection with other intervals, which your proof exploits but Ruzsa's paper does not explicitly record.)

      To prove his $L^2$ bounded basis result Ruzsa then chooses a suitable lacunary increasing set of random shifts of such $A$; this gluing together of lacunary blocks is as in your construction. I suspect that this 'glue together random translates' trick would also be an alternative route to your upper density result.

      As Ruzsa remarks 'if [the interval in $A+A$] were at the beginning (that is, if we had $A+A\supset [0,cN]$), then we could combine these sets to form a basis with bounded $1_A\ast 1_A(n)$'.

  • GPT 5.4 Thinking has located some relevant partial progress / nearby literature here:

    1. Lower-density obstruction (Kiss–Sándor (2021)): for Sidon sets $A$, it is known that $d(A+A)=0$. This implies that $\underline d(A+A)=0$. So the problem is false in the strongest bounded-representation regime (essentially $1_A*1_A(n)\leq 2$, up to ordered/unordered normalisation).

    2. Upper-density obstruction (Konstantoulas (2013)): if $\overline d(\mathbb N\setminus(A+A))<1/10$, then $1_A*1_A(n)>5$ for infinitely many $n$. Equivalently, if $1_A*1_A(n)\leq 5$ for all $n$, then $\overline d(A+A)\leq 9/10$. So one cannot get arbitrarily close to full upper density under the very small uniform bound $5$.

    3. Nearby result (Ruzsa (1990)): Ruzsa constructed a basis $A\subseteq\mathbb N$ of order $2$ with representation function bounded in square mean: $\sum_{n\leq N}(1_A*1_A(n))^2=O(N)$. Since $A$ is a basis, $A+A$ has both lower and upper asymptotic density $1$. However, as this is an $L^2$-type result, this does not seem to bear directly on the $L ^ \infty$ question here.

      • One remark in this direction: there seems to be a qualitative difference between posing this question for $1_A * 1_A$ and for $1_A * 1_{-A}$, despite these two functions having identical $L^2$ norms. Indeed, the analogous question for $1_A * 1_{-A}$ is false, even if one removes the obvious degeneracy at the origin. Sketch of proof: if $A+A$ has large lower density but $1_A * 1_{-A}$ stays bounded away from the origin, then easy counting arguments show that $|A \cap [1,M]| \asymp \sqrt{M}$ for all large $M$, and thus $|A \cap [\varepsilon M, M]| \asymp \sqrt{M}$ for some small $\varepsilon$. A further Cauchy-Schwarz then shows that for any $M^{1/2} \lll R \lll M$, the number of pairs $a,a' \in A \cap [\varepsilon M,M]$ with $1 \leq |a-a'| \leq R$ is $\gg R$. Summing over about $\log R$ many choices of $M$ and using the pigeonhole principle, we conclude that $1_A * 1_{-A}$ attains a value of $\gg \log R$ somewhere in $[1,R]$, a contradiction.

        This also makes it less likely that finite field constructions can resolve these issues, as they are not all that great at distinguishing $1_A * 1_A$ from $1_A * 1_{-A}$. (For instance they can produce Sidon sets, for which both $1_A*1_A$ and $1_A*1_{-A}$ are as spread out as possible.)
