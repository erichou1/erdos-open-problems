Does every convex polygon have a vertex with no other $4$ vertices equidistant from it?
#97: [Er46b][Er61][Er75f,p.100][Er87b,p.175][Er90][Er92e][Er95][Er97e]

Erdős originally conjectured this (in [Er46b]) with no $3$ vertices equidistant, but Danzer found a convex polygon on 9 points such that every vertex has three vertices equidistant from it (but this distance depends on the vertex). Danzer's construction is explained in [Er87b]. Fishburn and Reeds [FiRe92] have found a convex polygon on 20 points such that every vertex has three vertices equidistant from it (and this distance is the same for all vertices).

If this fails for $4$, perhaps there is some constant for which it holds? In [Er75f] Erdős claimed that Danzer proved that this false for every constant - in fact, for any $k$ there is a convex polygon such that every vertex has $k$ vertices equidistant from it. Since this claim was not repeated in later papers, presumably Erdős was mistaken here.

Erdős suggested this as an approach to solve [96]. Indeed, if this problem holds for $k+1$ vertices then, by induction, this implies an upper bound of $kn$ for [96].

The answer is no if we omit the requirement that the polygon is convex (I thank Boris Alexeev and Dustin Mixon for pointing this out), since for any $d$ there are graphs with minimum degree $d$ which can be embedded in the plane such that each edge has length one (for example one can take the $d$-dimensional hypercube graph on $2^d$ vertices). One can then connect the vertices in a cyclic order so that there are no self-intersections and no three consecutive vertices on a line, thus forming a (non-convex) polygon.

  • I believe I have a proof for this problem. The general case can be conditionally reformulated/reduced to the exact n=9 case.

    The claim: The following three statements together reduce Problem 97 to the case \(|P|=9\).

    1. Counting obstruction. Any counterexample \(P\) has \(|P| \ge 9\).

    2. Surplus removable-vertex package. Suppose \(P\) is a counterexample with \(|P| \ge 10\). Fix a Moser triangle \(T \subseteq P\) and its three opposite caps \(C_1, C_2, C_3\). A closed cap-sum identity gives\[ |C_1| + |C_2| + |C_3| \;=\; |P| + 3 \;\ge\; 13, \]so some opposite cap has size \(\ge 5\). From any opposite cap of size \(\ge 5\) one can extract a vertex \(v \in P\) whose deletion preserves the counterexample property.

    3. Strict-convex convention. Read "convex polygon" as "finite set of extreme vertices in strict convex position." (The alternative reading that allows redundant \(180^\circ\) boundary vertices needs a separate, routine degenerate-interface reduction.)

    Theorem (conditional). Under (1), (2) and (3), Problem 97 is equivalent to the single finite statement: no \(9\)-point set in strict convex position is a counterexample.

    So, the problem reduces to finding a 9-point counterexample in strict convex position, or proving the entire 9-point space does not contain a counterexample. I currently have axiom and sorry-free lean code proving this for n=9, modulo the actual computational results, which are taking a while to complete. Will share as soon as it's ready.

  • A small observation on the \(k=4\) case. Suppose every vertex \(v_i\) of a convex \(n\)-gon has some distance occurring at least four times among the other vertices. Choose four such vertices for each \(i\), and call the resulting set \(Q_i\). Let \(d(v)\) be the number of sets \(Q_i\) containing the vertex \(v\). Then \(\sum_v d(v)=4n\). For distinct centers, the corresponding circles are distinct and hence intersect in at most two points, so \(|Q_i\cap Q_j|\le 2\). Therefore \(\sum_{i<j}|Q_i\cap Q_j|=\sum_v \binom{d(v)}{2}\). By Cauchy--Schwarz, \(\sum_v d(v)^2\ge \frac{(4n)^2}{n}=16n\), and hence \(\sum_v \binom{d(v)}{2}=\frac12\left(\sum_v d(v)^2-\sum_v d(v)\right)\ge 6n\). On the other hand, \(\sum_{i<j}|Q_i\cap Q_j|\le 2\binom n2=n(n-1)\). Thus \(6n\le n(n-1)\), so any counterexample must satisfy \(n\ge 7\). In the extremal case \(n=7\), equality would force \(d(v)=4\) for every vertex and \(|Q_i\cap Q_j|=2\) for every pair of chosen circles. Thus the incidence matrix has the structure of a \(2-(7,4,2)\) design. I have not closed the geometry of this case; computational checks of the cyclic orderings suggest the remaining cases are highly constrained, but this is only evidence, not a proof.
