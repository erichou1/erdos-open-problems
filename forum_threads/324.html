Does there exist a polynomial $f(x)\in\mathbb{Z}[x]$ such that all the sums $f(a)+f(b)$ with $a<b$ nonnegative integers are distinct?
#324: [ErGr80,p.53]
number theory | powers | sidon sets

In other words, the set $\{ f(n) : n\geq 1\}$ should be a Sidon set.

Erdős and Graham describe this problem as 'very annoying'. Probably $f(x)=x^5$ should work. The Lander, Parkin, and Selfridge conjecture would imply that $f(x)=x^n$ has this property for all $n\geq 5$.

Ruzsa [Ru01b] has proved there exists $c\in [0,1]$ and $n_0\geq 1$ such that\[\{ n^5 +\lfloor cn^4\rfloor : n \geq n_0\}\]is a Sidon set.

It is easy to check that a quadratic $f$ cannot have this property. Dubickas and Novikas [DuNo21] have shown that a cubic $f$ cannot have this property. It is a classical fact that $x^4$ cannot have this property.

This is discussed in problems C9, D1, and F30 of Guy's collection [Gu04].

  • A trivial remark: if one can construct such an $f$ which is only asymptotically Sidon (no collisions for $a,b$ large enough), then one can remove all such collisions by replacing $f(x)$ with $f(qx+r)$ for a suitable arithmetic progression $\{qx+r: x \in {\bf Z} \}$ that avoids all the collisions.

    It is conceivable that one may be able to solve this problem assuming the Bombieri-Lang conjecture, in this spirit of this paper. If the integer (or rational) solutions to the three-dimensional variety $\{ (a,b,c,d): f(a)+f(b)=f(c)+f(d)\}$ (which presumably will be of general type for typical $f$) are trapped inside two-dimensional subvarieties, then one may be able to escape the integer points on these subvarieties by replacing $f$ with $f \circ Q$ for various polynomials $Q$.

    Siegel's theorem on integral points should be able to manage any one-dimensional family of solutions, so the main problems are with the two-dimensional families. The key algebraic geometry question seems to be the following: let $S$ be an irreducible surface in $\{ (a,b,c,d): f(a)+f(b)=f(c)+f(d)\}$ such that the covers $S_Q = \{(a',b',c',d'): (Q(a'),Q(b'),Q(c'),Q(d')) \in S \}$ are not of general type for any polynomial $Q$. What does this imply about $S$?

  • This problem has often been discussed in the context of Sidon sets. For more information, see F30, D1 and C9 in Unsolved Problems in Number Theory by Richard Guy.

    The best result so far is due to Ruzsa where he proved the existence of a real $\xi \in [0,1]$ for which the set
    $$\left \{n^5 + \left\lfloor \xi n^4\right\rfloor : n\ge n_0 \right \}$$
    is a Sidon set for a suitable $n_0$.

    It is easy to show that a linear or a quadratic polynomial cannot generate a Sidon set. Dubickas and Novikas recently proved that a cubic polynomial doesn't generate a Sidon set either. As far as I know, it is not known whether a quartic polynomial can generate a Sidon set.
