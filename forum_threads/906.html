OPEN This is open, and cannot be resolved with a finite computation.
Is there an entire non-zero function $f:\mathbb{C}\to \mathbb{C}$ such that, for any infinite sequence $n_1<n_2<\cdots$, the set\[\{ z: f^{(n_k)}(z)=0 \textrm{ for some }k\geq 1\}\]is everywhere dense?
#906: [Er56d][Er82e,p.72]
analysis | iterated functions
The open status of this problem reflects the current belief of the owner of this website.   • I would like to propose a probabilistic solution to the non-trivial, transcendental-entire-function interpretation of Problem 906.

    The literal formulation is trivial if polynomials are allowed, since every sufficiently high derivative of a non-zero polynomial is identically zero. So I address the natural version in which f is required to be transcendental entire.

    First, observe that the stated property is equivalent to the following cofinite hitting property:

    For every non-empty open set U ⊂ C, there exists N(U) such that f^(n) has at least one zero in U for every n ≥ N(U).

    Indeed, if this fails for some U, then infinitely many derivatives have no zero in U, and choosing those derivative orders gives an infinite subsequence whose union of zero sets is not dense. Conversely, if the cofinite property holds, then every infinite subsequence eventually contains an order n large enough to hit any prescribed U.

    Now consider the planar Gaussian entire function

    F(z) = Σ_{k≥0} ξ_k z^k / sqrt(k!),

    where the ξ_k are independent standard complex Gaussian random variables. Almost surely, F is entire and transcendental.

    For the n-th derivative F^(n), the covariance kernel is

    K_n(z,w) = ∂_z^n ∂_{\bar w}^n exp(z \bar w),

    and on the diagonal,

    K_n(z,z) = n! exp(|z|^2) L_n(-|z|^2),

    where L_n is the Laguerre polynomial.

    By the Edelman-Kostlan formula, the expected zero counting measure μ_n of F^(n) satisfies

    dμ_n(z) = (1 / 4π) Δ log K_n(z,z) dA(z).

    Using the Plancherel-Rotach asymptotic for Laguerre polynomials at negative argument, uniformly on compact annuli avoiding 0,

    log L_n(-|z|^2) = 2 sqrt(n)|z| - |z|^2/2 + O(log n).

    Therefore, on compact sets avoiding 0,

    log K_n(z,z) = C_n + 2 sqrt(n)|z| + |z|^2/2 + O(log n),

    where C_n is independent of z. Hence, for any non-negative non-zero test function φ ∈ C_c^2(D), with D a disk whose closure avoids 0,

    E[∫ φ dn_{F^(n)}]
    = ∫ φ dμ_n
    ≥ c_φ sqrt(n)

    for all sufficiently large n. This follows by integrating by parts, since away from 0,

    Δ|z| = 1/|z|,

    so

    ∫ |z| Δφ dA = ∫ φ / |z| dA > 0.

    Now apply an Offord-Sodin type estimate for linear statistics of zeros of Gaussian analytic functions. For a Gaussian analytic function g and φ ∈ C_c^2(C), one has a bound of the form

    P(|∫ φ(dn_g - dμ_g)| ≥ λ) ≤ 3 exp(-c λ / ||Δφ||_1).

    If F^(n) has no zero in D, then ∫ φ dn_{F^(n)} = 0. Since the expectation is at least c_φ sqrt(n), the Offord estimate gives

    P(F^(n) has no zero in D) ≤ C_D exp(-c_D sqrt(n)).

    This is summable in n. Hence, by Borel-Cantelli, for each fixed disk D whose closure avoids 0, almost surely F^(n) has a zero in D for all sufficiently large n.

    Finally, take a countable base of rational disks whose closures avoid 0. This still suffices for density, since every non-empty open set contains such a disk. Intersecting the corresponding probability-one events over this countable base, we obtain a realization f of the planar Gaussian entire function such that every non-empty open set contains zeros of f^(n) for all sufficiently large n.

    By the cofinite reformulation, this f satisfies the original condition for every infinite subsequence of derivative orders. Since f is transcendental entire almost surely, this gives an affirmative answer to the transcendental version of Problem 906.

    I have also prepared a full PDF note with the details and references. I would be grateful if others could check the argument, especially the use of the Laguerre asymptotic and the Offord-Sodin estimate in this setting.

    • There seems to be a gap in the proof posted above.
      The step claiming that for sufficiently large \(n\),\[
      |g(z)| < |n!c_n| \quad \text{for all } |z|<R
      \]does not follow from uniform convergence and is generally false.
      A concrete counterexample is \(f(z)=e^{z^2}\). We have\[
      f(z)=\sum_{j=0}^{\infty}\frac{z^{2j}}{j!},\qquad
      c_{2j}=\frac1{j!},\quad c_{2j+1}=0.
      \]For even orders \(n=2k\),\[
      f^{(n)}(z)=n!c_n+\sum_{m\ge1} n!\binom{n+m}{n}c_{n+m}z^{\,m}.
      \]The coefficient ratio of the \(z^2\)-term to the constant term equals\[
      \frac{n!\binom{n+2}{n}c_{n+2}}{n!c_n}
      =\frac{(2k+2)(2k+1)}{2(k+1)}=2k+1.
      \]Hence for any fixed radius \(R>0\),\[
      \frac{|\,\text{$z^2$ term}\,|}{|\,\text{constant term}\,|}
      =(2k+1)R^2\to\infty\quad(k\to\infty).
      \]Therefore\[
      \sup_{|z|=R}|g(z)|
      \ge (2k+1)R^2|n!c_n|>|n!c_n|
      \]for all sufficiently large even \(n=2k\). Thus the inequality \(|g(z)|<|n!c_n|\) fails, and the Rouché comparison used in that argument cannot be justified.

  • The webpage states: ''Erdős [Er82e] writes that this was solved in the affirmative 'more than ten years ago', but gives no reference or indication who solved it. From context he seems to attribute this to Barth and Schneider [BaSc72], but this paper contains no such result.'' I agree with this assessment. Below I make a short clarification.

    First, one easily checks that Problem #906 is equivalent to the following formulation:

    **Problem 2**. Does there exist a transcendental entire function $f(z)$ such that for every nonempty open set $U\subset\mathbb{C}$ there is an integer $N(U)>0$ with the property that, for all $n\ge N(U)$, the $n$-th derivative $f^{(n)}$ has a zero in $U$?

    After a careful reading of Barth and Schneider [BaSc72], I found that the methods in [BaSc72] only yield the following weaker positive result for *Problem 2*. I state it here as Proposition 1:

    **Proposition 1**. There exists a nonzero transcendental entire function $f(z)$ such that for every nonempty open set $U\subset\mathbb{C}$ there are infinitely many integers $n$ for which $f^{(n)}$ has a zero in $U$.

    Proof of Proposition 1. Let $S_k:=\frac{1}{k}\mathbb{Z}+\frac{i}{k}\mathbb{Z}$ denote the square lattice of mesh $1/k$. By [BaSc72, Theorem 1], there exist a sequence of positive integers $(n_k)$ and an entire function $f$ such that\[
    f^{(n_k)}(z)=0 \quad\text{for every } z\in S_k,\qquad k=1,2,\dots.
    \]Given any nonempty open set $U\subset\mathbb{C}$, there exists $K(U)\in\mathbb{N}$ such that $U\cap S_k\neq\emptyset$ for all $k\ge K(U)$ (since the lattices $S_k$ become dense as $k\to\infty$). Hence, for every $k\ge K(U)$ the function $f^{(n_k)}$ has a zero in $U$. This produces infinitely many integers $n$ with the required property, proving the proposition.

    In particular, [BaSc72] yields the weaker density conclusion that the union $\bigcup_{n\ge1}\{z:\,f^{(n)}(z)=0\}$ is dense in $\mathbb C$, but it does not establish the stronger uniform property required in Problem #906.

    • In addition to the weaker positive result from Barth--Schneider [BaSc72] (cf. Proposition 1 above), I found two further relevant results in the paper of Boas--Reddy [BoRe73]:
      R.~P. Boas Jr. and A.~R. Reddy, Zeros of successive derivatives of entire functions, Journal of Mathematical Analysis and Applications 42(1973), 466--473.

      By Theorem 1 in [BoRe73], we know that for entire functions of at most order $2$ and finite type, there is no function $f$ satisfying the property in Problem 2 above.

      By Theorem 2 in [BoRe73], if $p>2$, then there exists an entire function of order $p$ and a constant $A>0$ such that every disk of radius $A$, anywhere in the plane, contains a zero of every derivative $f^{(i)}$.

      These two theorems show that Problem 2 cannot be solved for functions of small order. Once the order exceeds $2$, one can indeed find entire functions with particular zero-distribution properties, but this still does not resolve Problem 2.

    • One more remark. I found a new proof of the *Proposition 1* above. Let \(H(\mathbb{C})\) be the space of all entire functions on $\mathbb{C}$ with the compact open topology. By a result of MacLane (see Theorem 6 in [Ma52] or Theorem 2 in [ArMa02]), there exists an entire function \(f\) such that the set \(\{f^{(n)}: n\in\mathbb{N}\}\) is dense in \(H(\mathbb{C})\). From this one obtains the density of the union of zero sets of the derivatives via a short Rouché-argument, as follows.

      Fix any point \(z_0\in\mathbb{C}\) and radius \(r>0\), and set the closed disk \(K=\overline{D(z_0,r)}\). Let \(g(z)=z-z_0\). Note that \(|g(z)|=r\) on \(\partial D(z_0,r)\). Since \(\{f^{(n)}\}\) is dense in \(H(\mathbb{C})\), there exists \(N\) with\[
      \sup_{z\in K}\,|f^{(N)}(z)-g(z)|<r.
      \]Restricting to \(\partial D(z_0,r)\) gives \(|f^{(N)}(z)-g(z)|<|g(z)|\). By Rouché’s theorem, \(f^{(N)}\) and \(g\) have the same number of zeros in \(D(z_0,r)\). Since \(g\) has exactly one zero at \(z_0\), the function \(f^{(N)}\) has a zero in \(D(z_0,r)\). Because \(z_0\) and \(r\) were arbitrary, we conclude that\[
      \bigcup_{n\ge 1}\{z\in\mathbb{C}:\ f^{(n)}(z)=0\}
      \]is dense in \(\mathbb{C}\). This completes the proof of Proposition 1.

      The above discussion naturally leads to:

      **Problem 3**. Does there exist \(f\in H(\mathbb{C})\) such that for every infinite subset \(S\subset\mathbb{N}\) the set \(\{D^{n}f : n\in S\}\) is dense in \(H(\mathbb{C})\)?

      If Problem 3 were true, it would give an affirmative solution to Problem #906. However, *Problem 3* is in fact false. Here is a short proof.

      Proof that Problem 3 is false. Let \(U,V\subset H(\mathbb C)\) be two disjoint nonempty open sets. Assume there exists \(f\in H(\mathbb C)\) such that for every infinite \(S\subset\mathbb N\) the set \(\{D^{n}f:n\in S\}\) is dense in \(H(\mathbb C)\). For any nonempty open \(W\subset H(\mathbb C)\) define\[
      N_W:=\{\,n\in\mathbb N:\ D^{n}f\in W\,\}.
      \]Then necessarily \(S\cap N_W\neq\varnothing\). Indeed, if \(N_W\) were not cofinite, taking \(S=\mathbb N\setminus N_W\) (which is infinite) would force \(\{D^{n}f:n\in S\}\subset H(\mathbb C)\setminus W\), contradicting its density. Hence both \(N_U\) and \(N_V\) are cofinite, so \(N_U\cap N_V\neq\varnothing\). Pick \(n\in N_U\cap N_V\), then \(D^{n}f\in U\cap V=\varnothing\), a contradiction. Therefore no such \(f\) exists.

      References.
      [Ma52] MacLane, Gerald R. "Sequences of derivatives and normal families." Journal d’Analyse Mathématique 2.1 (1952): 72-87.
      [ArMa02] ARON, RICHARD, and DINESH MARKOSE. "ON UNIVERSAL FUNCTIONS OCTOBER 29, 2002."

      In view of the above, a negative resolution of Problem #906 appears plausible, yet this observation is merely suggestive and falls well short of a proof.
