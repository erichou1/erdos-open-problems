For any $0<\alpha<1$, let\[f(\alpha,n)=\frac{1}{\log n}\sum_{1\leq k\leq n}(\tfrac{1}{2}-\{ \alpha k\}).\]Does $f(\alpha,n)$ have an asymptotic distribution function?

In other words, is there a non-decreasing function $g$ such that $g(-\infty)=0$, $g(\infty)=1$,
and\[\lim_{n\to \infty}\lvert \{ \alpha\in (0,1): f(\alpha,n)\leq c\}\rvert=g(c)?\]
#1002: [Er64b]
analysis | diophantine approximation

Kesten [Ke60] proved that if\[f(\alpha,\beta,n)=\frac{1}{\log n}\sum_{1\leq k\leq n}(\tfrac{1}{2}-\{\beta+\alpha k\})\]then $f(\alpha,\beta,n)$ has asymptotic distribution function\[g(c)=\frac{1}{\pi}\int_{-\infty}^{\rho c}\frac{1}{1+t^2}\mathrm{d}t,\]where $\rho>0$ is an explicit constant.

  • The literature here is somewhat confusing because the results are very sensitive to what is being averaged here: one can work with "temporal" averaging (averaging in $n$), "spatial" averaging (averaging in $\beta$), "quenched" averaging (fixed $\alpha$), and "annealed" averaging (averaging in $\alpha$). In particular, Erdos's question involves only the annealed notion of average. The most recent paper I could find (with the help of Gemini and ChatGPT, whose outputs were somewhat unreliable due to the subtleties mentioned above) was a 2024 paper of Fruhworth-Hauke, which seem to indicate that the Erdos problem remains open as stated, although with additional averaging the claim is true (e.g., if one also averages in $n$ the claim was obtained by Dolgopyat and Sarig).

    The problem here seems remarkably delicate, and probably the first step to anyone interested in trying to solve this problem is to carefully go through the literature (of which the above two papers are only a small sample).

    • Some notes on the problem (all familiar to experts on this problem, I assume, but it was somewhat novel to me). I was initially rather confused as to the logarithmic Cauchy behavior here, since square root cancellation heuristics would naively $f(\alpha,n)$ should oscillate like $\sqrt{n}$ times a Gaussian instead of $\log n$ times a Cauchy distribution. But it turns out the series is extremely subrandom, and the behavior of $f(\alpha,n)$ is closely tied with how $n$ interacts with the successive approximants $p_j/q_j$ of $\alpha$ in the continued fraction expansion, as per the Denjoy-Koksma inequality. Suppose for instance that $q_j \leq n < q_{j+1}$ with $j$ even, then by standard continued fraction theory we have $q_{j+1} = a_{j+1} q_j + q_{j-1}$ with $a_{j+1}$ the $j+1^{th}$ term in the continued fraction expansion (which would be expected to be distributed according to the Gauss-Kuzmin law, which is a discrete Cauchy distribution, and $\alpha = \frac{p_j}{q_j} + \delta$ for some $0 < \delta < \frac{p_{j+1}}{q_{j+1}}-\frac{p_j}{q_j} = \frac{1}{q_j q_{j+1}}$). One can then write $n = c_{j+1} q_j + n'$ for some $0 \leq c_{j+1} \leq a_{j+1}$ and $n' < q_j$ (if $c_{j+1}=a_{j+1}$ one can improve the upper bound on $N'$ to $N' < q_{j-1}$); the sequence of $c_{j+1}$ represent a sort of "base $(q_j)_{j=1}^\infty$ expansion" of $n$ (e.g., if $\alpha$ is the golden ratio then this is essentially Fibonacci coding). A key point here is that if $k \leq n$ then $\alpha k$ has the same integer part as $p_j k / q_j$. As such one can write
      $$ f(\alpha,n) \log n = \sum_{k \leq n} (\frac{1}{2} - \{p_j k / q_j \} + p_j k / q_j - \alpha k) = \sum_{k \leq n} (\frac{1}{2} - \{p_j k / q_j \}) - \frac{\delta n(n+1)}{2}$$
      and similarly
      $$ f(\alpha,n') \log n' = \sum_{k \leq n'} (\frac{1}{2} - \{p_j k / q_j \}) - \frac{\delta n'(n'+1)}{2}.$$
      The function $\frac{1}{2} - \{p_j k / q_j \}$ is $q_j$-periodic and sums to $-\frac{1}{2}$ on each period, so
      $$ \sum_{k \leq n} (\frac{1}{2} - \{p_j k / q_j \}) = \sum_{k \leq n'} (\frac{1}{2} - \{p_j k / q_j \}) - \frac{c_{k+1}}{2}.$$
      Hence we obtain a recursion
      $$ f(\alpha,n) \log n = f(\alpha,n') \log n' + X_j$$
      where
      $$ X_j = \frac{\delta}{2} (n(n+1)-n'(n'+1)) - \frac{c_{j+1}}{2} = \frac{c_{j+1}}{2} (\delta q_j (n+n'+1) - 1) $$
      (I may have gotten some signs wrong here). There is a similar formula (with some signs changed) when $j$ is odd instead of even. Each $X_j$ can be calculated to be of size $O(a_{j+1})$ and is primarily determined by the "digit" $c_{j+1}$ in the expansion of $n$ (as well as the continued fraction coefficient $a_{j+1}$). There are $\asymp \log n$ relevant values of $j$ here, with the $a_{j+1}$ behaving like iid copies of the heavy-tailed Gauss-Kuzmin distribution, and so the sum generically should be dominated by $O(1)$ terms where $a_{j+1} \asymp \log n$, each of which contributes a term of size $O(\log n)$ (but there is a rare "major arc" event in which one of the terms is much larger, which seems to defeat most moment-based strategies for this problem). This is where the $\log n$ normalization comes from, and the Cauchy distribution presumably reflects some basin of attraction property that I did not work out carefully (it reminds me of how the Cauchy distribution shows up in the analysis of the Saint Petersburg paradox).

      To solve this problem, therefore, one needs to understand the behavior of the digits $c_{j+1}$ in the expansion of a large but fixed number $n$, with respect to the base $(q_j)_{j=1}^\infty$ generated by a random continued fraction, and with a particular focus on those rare digits where $a_{j+1}$ is as large as $\log n$. It seems that the most significant digits (where $q_j$ is not much smaller than $n$) can be relatively well understood, but there is some weird mixing process going on with the operation $n \mapsto n'$ of taking successive remainders with respect to the $b_{j+1}$ that I don't understand past the first few steps (it reminds me a little bit of the Collatz iteration, actually, which is unnerving for the purposes of actually proving anything rigorous).
