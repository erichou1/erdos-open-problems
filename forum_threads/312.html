Does there exist some $c>0$ such that, for any $K>1$, whenever $A$ is a sufficiently large finite multiset of positive integers with $\sum_{n\in A}\frac{1}{n}>K$ there exists some $S\subseteq A$ such that\[1-e^{-cK} < \sum_{n\in S}\frac{1}{n}\leq 1?\]
#312: [ErGr80]
number theory | unit fractions

  • I have a forthcoming arxiv preprint that shows, using Vjeko_Kovac's notation, $\varepsilon(A) \le \exp\left(-c\sqrt{K\log K}\right)$. A simpler version of the same argument can also be used to show that $\varepsilon(A) \le \exp(-c\sqrt{K})$, and it's probably short enough that I can summarize it here:

    Choose $S$ maximizing $R(S)\le 1$ (again using Vjeko_Kovac's notation) and write $R(S)=1-\varepsilon$, $N=\varepsilon^{-1}$, $x=\log N$, and put $B=A\setminus S$. Then $R(B)>K-1$. Also every denominator in $B$ is $<N$, since otherwise some remaining term would have size $\le\varepsilon$ and could be added to $S$, contradicting maximality. We next compress $B$: whenever $p\mid n$ and there are $p$ copies of $n$, replace those $p$ copies by one formal copy of $n/p$. This preserves reciprocal mass, and every subset sum in the compressed multiset lifts to an actual subset sum in $B$. Let $C$ be a terminal compressed multiset, with multiplicities $m_n$. Then $R(C)>K-1$, all denominators in $C$ are $<N$, and $m_n<P^-(n)$ for every $n$, where $P^-(n)$ is the least prime factor of $n$. Also $n=1$ cannot occur, since that would already give an exact sum $1$.

    We use the following activation lemma: Let $E$ be a collection of distinct denominators in $C$ with $\alpha/2<m_n/n\le\alpha$, let $T=|E|$, and define $D(E)=\max_{q\le e^{C_0x}}\#\{n\in E:n\mid q\}$. If $H(E):=\sum_{n\in E}m_n/n\gg D(E)\sqrt{\alpha x}+\alpha D(E)x$, then the bounded sums $\sum_{n\in E}u_n/n$, $0\le u_n\le m_n$, contain a value in $(1-e^{-2x},1)$. Since $e^{-2x}<\varepsilon$, this contradicts the choice of $S$.

    The proof is as follows; choose a parameter $s$, activate each denominator $n\in E$ independently with probability $s/T$, and, if $n$ is activated, choose a coefficient uniformly from an interval ${0,\ldots,r_n}$ with $r_n\asymp n/s$, chosen so that the random sum $X=\sum u_n/n$ has mean approximately $1$. This is possible as long as $s\gg 1/\alpha$, which ensures $r_n\le m_n$. The variance of $X$ is $\asymp 1/s$.

    Let $t$ denote the Fourier frequency, so the characteristic function is $\phi_X(t)=\mathbb E \left[e^{itX}\right]$. For small frequencies $|t|\ll s$, Taylor expansion gives Gaussian decay $|\phi_X(t)|\le \exp(-ct^2/s)$. For larger frequencies, write $z=t/(2\pi)$ and note that a denominator $n$ is nearly resonant only when $z$ is close to a multiple of $n$. More precisely, if $\operatorname{dist}(z,n\mathbb Z)\le R$, then $n$ divides one of $O(R+1)$ integers near $z$, so by definition of $D(E)$ we have $\#\{n\in E:\operatorname{dist}(z,n\mathbb Z)\le R\}\ll (R+1)D(E)$. Sorting the distances from $z$ to the lattices $n\mathbb Z$ gives high-frequency decay $\exp(-c\cdot\min\{s,T^2/(D(E)^2s)\})$. Optimizing $s$ gives exactly the threshold $H(E)\gg D(E)\sqrt{\alpha x}+\alpha D(E)x$. Fourier inversion with a smooth nonnegative test function supported in $(1-e^{-2x},1)$ then shows that some bounded coefficient sum lies in that interval.

    It remains to sum the forced bounds over all denominator types in $C$. For prime denominators, $D(E)\ll x/\log x$, since an integer $q\le e^{C_0x}$ has $O(x/\log x)$ prime factors. Summing the activation bound over dyadic $\alpha$ gives total prime mass $O(x^2)$. For composite denominators, group dyadically by a denominator scale $P\le n<2P$ and by a multiplicity scale $m_n\asymp u$. Compression gives $P^-(n)>u/2$, hence $u\ll\sqrt P$ for composites. Moreover each such $n$ has at most $r(u)=\lfloor \log(2P)/\log(u/2)\rfloor$ prime factors, and therefore $D(E)\ll (C(1+x/\log P))^{r(u)}$. Applying the activation lemma to these cells and summing the dyadic $u$-levels gives $R(C\cap [P,2P))\ll (1+x/\log P)^2+\log P$. Summing this over dyadic $P<N$ gives total composite mass $O(x^2)$. Therefore $K-1<R(C)\ll x^2$, so $x\gg\sqrt K$. Since $x=\log(1/\varepsilon)$, this proves $\varepsilon\le \exp(-c\sqrt K)$.

  • Does anyone know if Erdős and Graham posed precisely the exponential "gap" for a good reason?
    For a finite multiset of positive integers $A$ denote\[ R(A) := \sum_{n\in A}\frac{1}{n} \]and\[ \epsilon(A) := \min\Big\{ 1 – R(S) \,:\, S\subseteq A,\, R(S)\leq 1 \Big\}. \]The problem asks if inequality\[ \epsilon(A) \leq e^{- c R(A)} \]holds with some constant $c>0$.
    Here is a simple example which comes close to showing that this is the best possible. For a positive integer $N$ take\[ A = \bigcup_{p\text{ prime},\ p\leq N} \{\underbrace{p,p,\ldots,p}_{p-1\text{ times}} \}. \]Then\[ R(A) = \sum_{p\text{ prime},\ p\leq N} \frac{p-1}{p} \sim \pi(N) = (1+o(1)) \frac{N}{\log N}, \]while $R(S)\neq1$ for every sub-multiset $S$ of $A$, so\[ \epsilon(A) \geq \prod_{p\text{ prime},\ p\leq N} \frac{1}{p} = \frac{1}{N\#} = e^{-(1+o(1))N}. \]Since $N=(1+o(1))R(A)\log R(A)$, this shows\[ \epsilon(A) \geq e^{- (1+o(1)) R(A) \log R(A)} \]for this particular sequence of examples of multisets $A$. It is not clear to me if a better (e.g., exponential) lower bound is possible, even for these particular multisets $A$.
    The actual conjectured inequality (the upper bound on $\epsilon(A)$) is a completely different story (many people here would know better), but I first want to clarify to myself how "optimal" it is. :-)
