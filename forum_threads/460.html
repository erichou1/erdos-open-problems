OPEN This is open, and cannot be resolved with a finite computation.
Let $a_0=0$ and $a_1=1$, and in general define $a_k$ to be the least integer $>a_{k-1}$ for which $(n-a_k,n-a_i)=1$ for all $0\leq i<k$. Does\[\sum_{0<a_i< n}\frac{1}{a_i}\to \infty\]as $n\to \infty$? What about if we restrict the sum to those $i$ such that $n-a_j$ is divisible by some prime $\leq a_j$, or the complement of such $i$?
#460: [Er77c,p.64][ErGr80,p.91]
number theory

There are no solutions, partial or complete, claimed in the comments.
This question arose in work of Eggleton, Erdős, and Selfridge, who could prove that $a_k <k^{2+o(1)}$ for $k$ large enough depending on $n$, but conjectured that in fact $a_k\ll k\log k$ is true.

The problem above is from [Er77c]. This question is stated slightly differently in [ErGr80], which has $a_0=n$ instead of $a_0=0$ and $1\leq i<k$ instead of $0\leq i<k$. The material difference this makes is that the main formulation above has $(a_k,n)=1$ for all $k\geq 1$, which is not imposed in the second formulation. Furthermore, in both [Er77c] and [ErGr80] this is stated without the restriction to $a_k<n$ in the sum, although perhaps this was implicitly intended. Without this condition the sum is infinite since the sequence of $a_k$ contains $n+p$ for all primes $p>n$ (this observation was made by Svyable using ChatGPT).

Unfortunately, although both sources mention a forthcoming paper of Eggleton, Erdős, and Selfridge, I cannot find a candidate paper of theirs with this problem in, and hence the motivation behind this problem, and what the precise problem intended was, is unclear.

Chojecki has noted in the comments that a positive solution to the main problem would follow if\[f(n) = \sum_{a<n}1_{P^-(n-a)>a}\frac{1}{a}\to \infty,\]where $P^-(\cdot)$ is the least prime factor. Standard estimates on rough numbers show that $\frac{1}{N}\sum_{n\leq N}f(n)\gg \log\log N$, so $f(n)$ does diverge on average, but it is unclear whether $f(n)\to \infty$ for all $n$.

  • This problem is stated differently in [Er77c] and [ErGr80]. An easier way to phrase the difference is that in [Er77c] we have $(a_k,n)=1$ for all $k\geq 1$ but this is not guaranteed in the [ErGr80] formulation. For example with $n=6$ the [Er77c] sequence begins $0,1,5$ while the [ErGr80] sequence begins $6,1,2$.

    As written, the sum trivially diverges, since it contains $a_k=n+p$ for all large primes $p$. This is the trivial solution posted on Reddit that natso26 mentions.

    Perhaps a condition like $a_k\leq n$ was intended, but neither source mentions this, and without any further reference or motivation given for this problem I don't think we can resolve what was actually meant.

    The proof posted by Przemek Chojecki would resolve this stronger version if $f(n)=\sum_{a<n} \frac{1}{a}1_{P^-(n-a)>a}\to \infty$. Averaging this over $n\leq N$ and using estimates on rough numbers show (I think) that it grows like $\log\log n$ on average. It's not clear to me whether this $\to \infty$ along all $n$ though.

    • You're right, in either formulation, if one sums over all $k$ without any upper cut-off on $a_k$, then the series is automatically divergent: for every prime $p>n$ the term $a=n+p$ must occur in the greedy sequence, because before reaching $n+p$ all previously chosen translates satisfy $n-a_i\in[-p+1,n-1]$, so none is divisible by $p$ (the only multiple of $p$ in that interval is $0$, which never arises as $n-a_i$ in these constructions); hence $\gcd(n-(n+p),\,n-a_i)=\gcd(p,\,n-a_i)=1$ for all earlier $i$, making $n+p$ admissible at the moment it becomes available and therefore impossible for the greedy rule to skip. Comparing $\sum_{p>n}1/(n+p)$ with $\sum_p 1/p$ gives divergence for every fixed $n\ge2$.

      If instead one imposes the natural ''nontrivial'' restriction $a_k<n$ (or $a_k\le n$), then the above argument disappears and one can at least prove an averaged divergence statement from the forced ''rough'' contribution. Writing $P^-(m)$ for the least prime factor of $|m|$ (with $P^-(\pm1)=\infty$), set\[
      f(n):=\sum_{a=1}^{n-1}\frac1a\,\mathbf 1_{\{P^-(n-a)>a\}},
      \]so that (by the forced-rough lemma) the truncated greedy sum $S_{<n}(n):=\sum_{a_k\le n-1}1/a_k$ satisfies $S_{<n}(n)\ge f(n)$ for every $n$ in both formulations. Averaging over $n\le N$ and changing variables $m=n-a$ yields $\sum_{n\le N}f(n)=\sum_{a\le N-1}\frac1a\,\Phi(N-a,a)$, where $\Phi(x,y)$ counts $y$-rough integers $\le x$; Buchstab’s asymptotic for $\Phi(x,y)$ (uniform for $a\in[N^{1/3},N^{1/2}]$, where $u=\log(N-a)/\log a\in[2,3]$ and hence $\omega(u)\gg1$) gives $\Phi(N-a,a)\gg N/\log a$ on that range, and thus\[
      \frac1N\sum_{n\le N}f(n)\ \gg\ \sum_{N^{1/3}\le a\le N^{1/2}}\frac1{a\log a}\ \asymp\ \log\log N.
      \]In particular $\limsup_{n\to\infty}f(n)=\infty$ and hence $\limsup_{n\to\infty}S_{<n}(n)=\infty$; what remains open is whether $f(n)\to\infty$ (and thus $S_{<n}(n)\to\infty$) along all $n$.

      I've re-written the note to discuss both cases and give the full proofs. I think this solves the problem. See here: https://www.ulam.ai/research/erdos-460-v2.pdf

  • As Kevin Barreto mentioned, we might have $a_0=0$ with $i=0$ included in $gcd$ condition. The full proof of both versions is here: https://www.ulam.ai/research/erdos-460.pdf - here's the sketch:

    Fix $n\ge2$ and define a greedy increasing sequence $(a_k)$ by requiring the translates $n-a_k$ to be pairwise coprime: in Case A one enforces $\gcd(n-a_k,n-a_i)=1$ for all $1\le i<k$, while in Case B one includes $i=0$ with $a_0=0$, which is equivalent to adding the standing constraint $\gcd(a_k,n)=1$. To study Erdős' divergence question for $\sum_k 1/a_k$ (and its natural subsums), the note introduces a dichotomy for $1\le a\le n-1$: call $a$ rough if the least prime factor $P^{-}(n-a)$ exceeds $a$ (equivalently, $n-a$ has no prime divisor $\le a$), and proper otherwise. This yields a canonical subsum $S_{\mathrm{rough}}(n)=\sum_{k:\,a_k\le n-1,\ P^{-}(n-a_k)>a_k}1/a_k$, with complementary ''proper'' subsum capturing the genuinely interactive part of the greedy dynamics.

    The main reduction is that the rough part is forced and admits an explicit description independent of the history: for any $a<n$ with $P^{-}(n-a)>a$, one has $\gcd(n-a,n-a')=1$ for every $a'<a$ (since any common prime divisor would divide $a-a'$ and hence be $\le a$), so such an $a$ is admissible at every stage once it becomes available (and in Case B this automatically includes coprimality with $n$). A greedy minimality argument then shows the process cannot skip any rough $a$: if $a$ were the least skipped rough integer, at the first step where $a_{k-1}<a<a_k$ it would be admissible, contradicting the definition of $a_k$ as the least admissible choice. Consequently the set of rough values appearing with $a_k<n$ is exactly $R(n)=\{1\le a\le n-1: P^{-}(n-a)>a\}$, and hence\[
    S_{\mathrm{rough}}(n)=\sum_{a\in R(n)}\frac1a=\sum_{\substack{1\le a\le n-1\\ P^{-}(n-a)>a}}\frac1a,
    \]giving an explicit, history-free lower bound for $\sum_k 1/a_k$ in both cases and reducing the divergence problem to estimating this rough sum plus the remaining proper contribution.
