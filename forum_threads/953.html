Let $A\subset \{ x\in \mathbb{R}^2 : \lvert x\rvert <r\}$ be a measurable set with no integer distances, that is, such that $\lvert a-b\rvert \not\in \mathbb{Z}$ for any distinct $a,b\in A$. How large can the measure of $A$ be?
#953: [Er77c]
geometry | distances

A problem of Erdős and Sárközi. Erdős [Er77c] writes that 'Sárközi has the sharpest results, but nothing has been published yet'.

The trivial upper bound is $O(r)$. Koizumi and Kovac have observed in the comments that Sárközy's lower bound in [466] can be adapted to give a lower bound of $\gg_\epsilon r^{1/2-\epsilon}$ for all $\epsilon>0$.

See also [465] for a similar problem (concerning upper bounds) and [466] for a similar problem (concerning lower bounds).

    M(R)\ll R^{1/2} \qquad (R\geq 1).

    • I don't want to delay this solution any further, so I can say that the proof generated by Przemek (with ChatGPT) is correct and the simplification generated by Nat (with ChatGPT) is expositionally quite satisfactory and does the job.

      The proof is in fact really nice and the main idea is simple. I like calling it Delsarte's method, so perhaps I swayed GPT to use that name. Approximating the set from below (using regularity of the Lebesgue measure) one can assume that $A$ is compact, in which case it doesn't only avoid integer distances between pairs of its points - it also avoids achieving distances that are $s$-close to an integer, for some $s>0$. One then defines a certain "witness function" $K_s$ (constructed as a weighted superposition of dilates of the Bessel function $J_0$) with properties:
      $K_s$ is radial and positive definite,
      $K_s(x) \lesssim s^{-2}$ for $|x|\leq s$,
      and $K_s(x) \lesssim - |x|^{-1/2}$ when $\| |x| \|_{\mathbb{Z}}\geq s$.
      Now,
      \begin{align*}
      & 0\leq \iint K_s(x-y) 1_A(x) 1_A(y) dx dy \\
      & = \iint_{|x-y|< s} \underbrace{K_s(x-y)}_{\lesssim s^{-2}} 1_A(x) 1_A(y) dx dy \\
      & + \iint_{\| |x-y| \|_{\mathbb{Z}}\geq s} \underbrace{K_s(x-y)}_{\lesssim R^{-1/2}} 1_A(x) 1_A(y) dx dy \\
      & + \underbrace{\iint_{|x-y|\geq s, \| |x-y| \|_{\mathbb{Z}}< s} K_s(x-y) 1_A(x) 1_A(y) dx dy}_{=0}
      \end{align*}
      which leads to\[ 0\leq Cs^{-2}s^2|A| - C'R^{-1/2} (|A|^2-s^2|A|), \]i.e., $|A|\lesssim R^{1/2}$ in the limit as $s\to0$.

      This is clearly a modification of Konyagin's solution of [465] "inspired" by Junnosuke's comment on how the problem relates to the discrete problem. I still feel a bit uncomfortable with a proof that performs an approximation argument up to some $s>0$ that plays no role in the final estimate, but it's correct, and that would be something to think about later. I would still prefer the proof along the lines of my comment above, which is how I would start working on the problem, but, hey, I didn't solve it that way, so maybe there's a good reason why this is better. :-)

    It is trivial to see that the order of growth is $O(r)$: just consider intersections of $A$ with translates of the horizontal integer lattice.
    In the other direction, one can use Sárközy's result on Problem [466] (taking the set $A$ to be a union of congruent disks) and obtain something like $\Omega(r^{0.26})$.
    I suspect that the true order of magnitude could be around $r^{1/2}$, but I was not able to modify Konyagin's beautiful solution of Problem [465] to work here as well.

    • Using Sárközy’s result, one obtains a stronger bound $\Omega(X^{1/2-\varepsilon})$. More precisely, Sárközy’s theorem asserts that for any sufficiently small $\delta>0$ and all $X > X_0(\delta)$, one has
      $$N(X,\delta) > X^{1/2-\delta^{1/7}},
      $$where $N(X,\delta)$ is the function defined in [465]. Taking $\delta=\varepsilon^7$, it suffices to place disks of radius $\delta/3$ centered at each point of Sárközy’s point set.

      As for the upper bound, if one works out the implied constant in Konyagin’s proof of the estimate $N(X,\delta)\ll_{\delta} X^{1/2}$ for [465], one obtains something like
      $$N(X,\delta) \delta^{5/2}\ll X^{1/2}.
      $$To apply this to the present problem, one would need to improve the left-hand side to $N(X,\delta) \delta^{2}$. However, I suspect that a better construction should exist.
