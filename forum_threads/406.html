Is it true that there are only finitely many powers of $2$ which have only the digits $0$ and $1$ when written in base $3$?
#406: [Er79,p.67][ErGr80,p.80]
number theory | base representations

The only examples seem to be $1$, $4=1+3$, and $256=1+3+3^2+3^5$. If we only allow the digits $1$ and $2$ then $2^{15}$ seems to be the largest such power of $2$.

This would imply via Kummer's theorem that\[3\mid \binom{2^{k+1}}{2^k}\]for all large $k$.

Saye [Sa22] has computed that $2^n$ contains every possible ternary digit for $16\leq n \leq 5.9\times 10^{21}$.

Let $N(x)$ count the number of $n\leq x$ such that $2^n$ has only the digits $0$ and $1$ in base $3$. Narkiewicz [Na80] proved\[N(x)\leq 1.62 x^{\log_32}\]This is mentioned in problem B33 of Guy's collection [Gu04]. There are many generalisations possible - see, for example, [AbLa14] and [La09].

  • As mentioned, the non-averaged version of the main result of "Bits of $3^n$ in binary, Wieferich primes and a conjecture of Erdos(2016)" by T. Dupuy and D. E. Weirich would imply this conjecture.

    Moreover, they mention that the progress on this conjecture has been done on this one by improving the bounds on the function $N(X)$, where $N(X) =$ {$\# n \leq X : (2^n)_3$ omits a $2$}
    (For a prime $q$ and a number $N$ we will let $(N)q$ denote the base $q$ expansion of $N$. We view a base $q$ expansion as a string of numbers from the set $\{0, 1, ..., q − 1\}$).

    The best known bound is due to Narkiewicz in [Na80]: $N(X) \leq 1.62X^{\alpha_0}$, where $\alpha_0= \log_3(2) \approx 0.630$.

    [Na80]: W. Narkiewicz, A note on a paper of H. Gupta concerning powers of 2 and 3, Publ. Elektroteh. Fak. Univ. Beogr., Ser. Mat. Fiz. 678–715 (1980) 173–174.
