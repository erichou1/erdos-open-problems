What is the size of the largest Sidon subset $A\subseteq\{1,2^2,\ldots,N^2\}$? Is it $N^{1-o(1)}$?
#773: [Er80,p.109][AlEr85]
number theory | sidon sets | squares

There are claims of partial results and/or a path to a solution in the comments.
A question of Alon and Erdős [AlEr85], who proved $\lvert A\rvert \geq N^{2/3-o(1)}$ is possible (via a random subset), and observed that\[\lvert A\rvert \ll \frac{N}{(\log N)^{1/4}},\]since (as shown by Landau) the density of the sums of two squares decays like $(\log N)^{-1/2}$. This upper bound was improved to\[\lvert A\rvert \leq N^{1-\frac{c}{\log\log N}}\]for some constant $c>0$ by Croot, Mao, and Yip (see the comments).

The lower bound was improved to\[\lvert A\rvert \gg N^{2/3}\]by Lefmann and Thiele [LeTh95].

In [Er80] Erdős further defines $g(A)$ to be the maximal size of a Sidon set of $\{a^2 : a\in A\}$, and asks whether $g(A)\geq g(\{1,\ldots,N\})$ where $N=\lvert A\rvert$. (See [530] for the linear analogue.)

He further asks the infinite analogue: is there an infinite set $A\subset \mathbb{N}$ such that $\lvert A\cap [1,N]\rvert \geq N^{1-o(1)}$ for all large $N$ and $\{a^2 : a \in A\}$ is a Sidon set?

See [1206] for a similar question for higher powers.

  • My student Junzhe Mao and postdoc Kyle Yip had worked on this problem some, and then Junzhe made an advance on it, and we have a "proof from the book" upper bound.  This and other results will be included in a forthcoming paper we are writing. Below is a version of the argument, which was heavily inspired by Brandon Hanson's paper "Additive Correlation and the Inverse Problem for the Large Sieve". We didn't use AI for this work.

    Basically, we have an upper bound of the form:
    $$
    |A|\ <\ N \exp \left ( - c {\log N \over \log\log N}\right ),
    $$
    for some $c > 0$.

    Proof: We can assume $|A| > N^{1-\varepsilon}$, or else we are done. Let $r_{A-A}(n)$ denote the number of ordered pairs $a,a'\in A$, $a \neq a'$, such that $a-a' =n$. Note that if $A$ is a Sidon set, it implies that $r_{A-A}(n) = 0$ or $1$. Let $d$ be an integer that is a product of about $t = (1-2\varepsilon)(\log N)/ \log\log N$ primes in $[\log N, 2 \log N]$; and note that the Prime Number Theorem implies $d < N^{1-2\varepsilon+o(1)}$, so $|A|>d$. Let $A(r,d)$ denote the set of elements of $A$ that are $r$ mod $d$.

    Now, $A$ is contained in the squares, and since the squares occupy $(p+1)/2$ residue classes mod $p$, for each prime $p \geq 3$, we have that $A$ can occupy at most $d(1+o(1)) / 2^t$ residue classes mod $d$. Using Cauchy-Schwarz we have
    $$
    \sum_{|n| \leq N^2 \atop d|n} r_{A-A}(n) = \left ( \sum_{r=0}^{d-1} |A(r,d)|^2 \right ) - |A| \gg {2^t |A|^2 \over d} - |A| > {2^{t-1} |A|^2 \over d},
    $$
    where the last inequality follows from the fact that $|A|>d$.

    On the other hand, since $A$ is Sidon (implying $r_{A-A}(n) =0$ or $1$, recall), the first sum is bounded from above by about $2N^2/d$. So, we get
    $$
    {2N^2 \over d} \gg {2^{t-1} |A|^2 \over d},
    $$
    and the result follows.

      • Just thought I would mention one more thing. See this paper (Theorem 1.2):

        https://onlinelibrary.wiley.com/doi/abs/10.1002/rsa.20496

        If the squares in $[N^2]$ behave like random sets of size $N$, then one would expect that the largest Sidon set contained within it has size about $N^{2/3 + o(1)}$. If so, then it means that improving the lower bound by finding a better construction is a fool's errand; and that what one *should* be doing is trying to improve the upper bound.

            It would interesting to try to come up with some nice algebraic or interesting quasi-algebraic set where the largest Sidon set inside it is larger than the random model would predict for a set of similar density -- in case the lower bound on #773 can be improved, yet the improvement is very hard to find. e.g. maybe something like the set of numbers that are the sum of two 4th powers, which seems like it might be close enough to being Sidon that all you need is a small tweak.
