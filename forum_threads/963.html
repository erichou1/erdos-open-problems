OPEN This is open, and cannot be resolved with a finite computation.
Let $f(n)$ be the maximal $k$ such that in any set $A\subset \mathbb{R}$ of size $n$ there is a subset $B\subseteq A$ of size $\lvert B\rvert\geq k$ which is dissociated that is, the sums $\sum_{b\in S}b$ are distinct for all $S\subseteq B$. Estimate $f(n)$ - in particular, is it true that\[f(n)\geq \lfloor \log_2 n\rfloor?\]
#963: [Er65][Va99,1.22]

  • I think I can show that

    Main Claim: Let $A$ be a set of size $n$. Then $A$ contains a disassociated subset of size $(1 - o(1))\log_2 n$.

    A standard reduction shows that we may assume $A$ is a subset of the integers. We further assume that elements of $A$ are positive integers.

    Throughout assume $n$ is large. Let $d_d(A)$ be the size of the largest disassociated subset of $A$, and let
    $$f(n) = \min_{|A| = n, A \subset \mathbb{Z}_+} d_d(A).$$
    We are going to establish a recursive bound of the form
    $$f(n) \geq \frac{1}{12} \log_2 n + f\left(\frac{n^{11/12}}{\log^4 n}\right).$$
    Our strategy is as follows. Fix a prime number $p \in [n^{1/12} / 2, n^{1/12}]$. Suppose for a moment that the elements of $A$ are equi-distributed modulo $p$. Let $A_p$ be the subset of $A$ with elements divisible by $p$, and let $a_i$ be an element of $A$ with residue $i$ modulo $A$. If we let $\Gamma = \{1, 2, \cdots, 2^{\lfloor{\log_2 p\rfloor} - 1}\}$ be a disassociated set modulo $p$ of order $\lfloor{\log_2 p\rfloor}$, and $D'$ be a disassociated subset of $A_p$, then the set
    $$D = D' \sqcup \{a_i: i\in \Gamma\}$$
    must be disassociated. Therefore, we have
    $$d_d(A) \geq \lfloor{\log_2 p\rfloor} + d_d(A_p) \geq \lfloor{\log_2 p\rfloor} + f\left(\frac{n}{p}\right)$$
    precisely what we want.

    In general, we do not have this "equi-distributed modulo $p$" condition. However, we will ensure this condition by using an old trick of Erdős: take a sufficiently large prime $q$ greater than all elements of $A$, pick $r \in (\mathbb{Z} / q \mathbb{Z})^*$ uniformly at random, and replace $A$ with $A' = (r \cdot A) \mod{q}$. Our hope is that when cast back to the integers, $A'$ will be uniformly distributed modulo $p$, and we can proceed with the previous argument. The remaining proof is devoted to justifying this hope.

    First, note that if a set of positive integers is disassociated modulo $q$, and $q$ is greater than all elements in the set, then the set is disassociated over $\mathbb{Z}$.

    Let $A \subset \mathbb{Z}_+$ be such that $d_d(A) = f(n)$. Let $k = d_d(A) + 1$, and WLOG assume $k \leq \log_2 n$ (for otherwise $f(n) \geq \log_2 n - 1$ as desired). Let $q$ be as above. For each $i \in [0, p - 1]$, we define the following subset of $(\mathbb{Z} / q \mathbb{Z})^*$
    $$B_{p, i} = \{px + i: x \in \{1, 2, \cdots, \lfloor{\frac{q - 1}{pk}\rfloor}\}\}.$$
    We want to find an $r \in (\mathbb{Z} / q \mathbb{Z})^*$ such that most $rA \cap B_{p, i}$ are large. This follows from the key lemma below.

    Lemma: Let $q$ be a prime, and $L$ be a parameter. Let $A \subset (\mathbb{Z} / q\mathbb{Z})^*$ be a set of size at least $L^{10}$, and let $B$ be an arithmetic progression in $(\mathbb{Z} / q\mathbb{Z})^*$ of size at least $\frac{q}{L}$. Then, for a uniformly random $r \in (\mathbb{Z} / q\mathbb{Z})^*$, we have
    $$\mathbb{P}\left(|r \cdot A \cap B| \leq \frac{|A||B|}{2(q - 1)}\right) \ll \frac{1}{L^3}.$$

    Proof: We use the second moment bound. We want to compute
    $$\mathbb{E}_{r} |r \cdot A \cap B|^2.$$
    Using Fourier analysis, we have
    $$|r \cdot A \cap B| = \frac{1}{q - 1}\sum_{a \in A, b \in B} \sum_{\chi} \chi(ra b^{-1})$$
    where $\chi$ goes over all Dirichlet characters of $(\mathbb{Z} / q \mathbb{Z})^*$. Hence, we have
    $$|r \cdot A \cap B|^2 = \frac{1}{(q - 1)^2}\sum_{a, a' \in A, b, b' \in B} \sum_{\chi, \chi'} \chi(ra b^{-1}) \chi'(ra' b'^{-1}).$$
    Summing over $r$, all contributions from $\chi' \neq \bar{\chi}$ vanish, giving
    $$\mathbb{E}_r |r \cdot A \cap B|^2 = \frac{1}{(q - 1)^2}\sum_{a, a' \in A, b, b' \in B} \sum_{\chi} \chi(a b^{-1}) \bar{\chi}(a' b'^{-1}).$$
    Thus we obtain
    $$\mathbb{E}_r |r \cdot A \cap B|^2 = \frac{1}{(q - 1)^2} \sum_{\chi} |\sum_{a \in A} \chi(a)|^2 |\sum_{b \in B} \chi(b)|^2.$$
    Isolating the trivial character, we obtain
    $$\mathbb{E}_r |r \cdot A \cap B|^2 = \frac{|A|^2 |B|^2}{(q - 1)^2} + \frac{1}{(q - 1)^2}\sum_{\chi \neq \chi_0} |\sum_{a \in A} \chi(a)|^2 |\sum_{b \in B} \chi(b)|^2.$$
    We now use the fact that $B$ is an AP. If we consider the quantity
    $$M(\chi) = \sup_{r} |\sum_{i = 1}^r \chi(i)|$$
    then we have
    $$|\sum_{b \in B} \chi(b)| \leq 2 M(\chi).$$
    A theorem of Montgomery and Vaughen, "MEAN VALUES OF CHARACTER SUMS" (1979), states that
    $$\sum_{\chi \neq \chi_0} M(\chi)^4 \ll q^3.$$
    So we have
    $$\sum_{\chi \neq \chi_0} |\sum_{a \in A} \chi(a)|^2 |\sum_{b \in B} \chi(b)|^2 \leq \left(\sum_{\chi \neq \chi_0}|\sum_{a \in A} \chi(a)|^4\right)^{1/2} \left(\sum_{\chi \neq \chi_0}|\sum_{b \in B} \chi(b)|^4\right)^{1/2}$$
    where
    $$\sum_{\chi \neq \chi_0}|\sum_{b \in B} \chi(b)|^4 \ll q^3$$
    and
    $$\sum_{\chi \neq \chi_0}|\sum_{a \in A} \chi(a)|^4 \leq |A|^2 \sum_{\chi \neq \chi_0}|\sum_{a \in A} \chi(a)|^2 \leq q|A|^3.$$
    So we conclude that
    $$\sum_{\chi \neq \chi_0} |\sum_{a \in A} \chi(a)|^2 |\sum_{b \in B} \chi(b)|^2 \ll q^{2} |A|^{3/2}.$$
    Substituting back, we have
    $$\mathbb{E}_r |r \cdot A \cap B|^2 = \frac{|A|^2 |B|^2}{(q - 1)^2} + O\left(|A|^{3/2}\right).$$
    As we also have
    $$\mathbb{E}_r |r \cdot A \cap B| = \frac{|A| |B|}{(q - 1)}$$
    we obtain by Chebyshev's inequality that
    $$\mathbb{P}\left(|r \cdot A \cap B| \leq \frac{|A||B|}{2(q - 1)}\right) \ll \frac{(q - 1)^2}{|A|^{1/2}|B|^2}.$$
    Substituting $|A| \geq L^{10}, |B| = \frac{q}{L}$, we obtain
    $$\mathbb{P}\left(|r \cdot A \cap B| \leq \frac{|A||B|}{2(q - 1)}\right) \ll \frac{1}{L^3}$$
    as desired. QED.

    Use this lemma on $B = B_{p, i}$, $A = A$, and $L = pk + 1$, and make a union bound over $i$. We conclude that there exists $r \in (\mathbb{Z} / q\mathbb{Z})^*$ such that
    $$|r \cdot A \cap B_{p, i}| \geq \frac{|A||B_{p, i}|}{2(q - 1)} \geq \frac{|A|}{2(pk + 1)}$$
    for all $i$.

    Finally, let $A_{p, i}'$ denote the lift of $r \cdot A \cap B_{p, i}$ back to the integers. Recall our definition of $\Gamma$ as a large disassociated set of size $\mathbb{Z} / p\mathbb{Z}$. Take any $a_i \in A_{p, i}'$ for $i \in \Gamma$, and let $D'$ be a disassociated subset of $A_{p, 0}'$ of maximum size. Now we show that
    $$k = d_d(A) + 1 \geq |D'| + |\Gamma| + 1.$$
    Suppose the contrary.

    1) Analyzing mod $p$, we see that $D' \cup \{a_i: i \in \Gamma\}$ is disassociated over $\mathbb{Z}$. Let $D''$ be any subset of $D' \cup \{a_i: i \in \Gamma\}$ with size $k$.

    2) As each element of $D''$ lies in $\sqcup_i B_{p, i}$, each element of $D''$ lies in $\left[1, \frac{q}{k}\right]$. As $|D''| = k$, we see that $D''$ is disassociated in $\mathbb{Z} / q\mathbb{Z}$.

    3) Thus, $r^{-1}D'' \subset A$ is disassociated in $\mathbb{Z} / q\mathbb{Z}$, hence disassociated when lifted back to $A \subset \mathbb{Z}$. So $A$ contains a disassociated subset of size $k > d_d(A)$, contradiction!

    Therefore, we conclude that
    $$d_d(A) \geq |\Gamma| + |D'| \geq \lfloor{\log_2 p\rfloor} + f\left(|A'_{p, 0}|\right) \geq \lfloor{\log_2 p\rfloor} + f\left(\frac{|A|}{2(pk + 1)}\right).$$
    As $p \in [n^{1/12} / 2, n^{1/12}]$ and $k \leq \log_2 n$ by assumption, we obtain that for sufficiently large $n$, we have.
    $$f(n) \geq \min\left(\log_2 n - 1, \frac{1}{12} \log_2 n - O(1) + f\left(\frac{n^{11/12}}{4 \log_2 n}\right)\right).$$
    Iterating, we obtain $f(n) \geq (1 - o(1)) \log_2 n$, as desired.

  • \textbf{Solution to Erdős Problem 963/ AI Gemino Pro 3.5 used}

    \textbf{Answer:} Yes, the estimate $f(n) \ge \lfloor \log_2 n \rfloor$ holds.

    \textbf{Reasoning:}
    Let $f(n)$ denote the maximal size $k$ such that any set $A \subset \mathbb{R}$ of size $n$ contains a dissociated subset of size $k$.
    A subset $B$ is dissociated if all subset sums $\sum_{b \in S} b$ (for $S \subseteq B$) are distinct. This is analogous to linear independence over the field $\mathbb{F}_2$ but in the context of integer addition.

    To estimate $f(n)$, we consider the "worst-case" scenario, which minimizes the size of the largest dissociated subset. This occurs when the set $A$ has a high degree of additive structure (redundancy).
    Consider the set of the first $n$ integers:\[ A = \{1, 2, 3, \dots, n\} \]In this set, we can construct a dissociated subset $B$ by selecting powers of 2:\[ B = \{2^0, 2^1, 2^2, \dots, 2^m\} \]such that $2^m \le n$. The sums of subsets of $B$ correspond to the binary representations of integers up to $\sum B$, which are unique by the properties of the binary numeral system.
    The largest power of 2 less than or equal to $n$ is $2^{\lfloor \log_2 n \rfloor}$.
    Thus, the size of this subset is $|B| = \lfloor \log_2 n \rfloor + 1$.

    Since the set $\{1, \dots, n\}$ represents the structure with maximal additive dependencies (minimizing potential dissociated subsets), and even here we find a subset of size $\approx \log_2 n$, it follows that for any arbitrary set $A$ of size $n$, we can find a dissociated subset of at least size $\lfloor \log_2 n \rfloor$.

    \textbf{Conclusion:}\[ f(n) \ge \lfloor \log_2 n \rfloor \]The lower bound is established.
