Let $d_n=p_{n+1}-p_n$. The set of $n$ such that $d_{n+1}\geq d_n$ has density $1/2$, and similarly for $d_{n+1}\leq d_n$. Furthermore, there are infinitely many $n$ such that $d_{n+1}=d_n$.
#218: [Er55c][Er57][Er61][Er65b][Er85c]
number theory | primes

There are no solutions, partial or complete, claimed in the comments.
In [Er85c] Erdős also conjectures that $d_n=d_{n+1}=\cdots=d_{n+k}$ is solvable for every $k$ (which is equivalent to $k$ consecutive primes in arithmetic progression, see [141]).

Banks [Ba23] has given a heuristic argument, assuming a quantitative form of the prime tuples conjecture, that the first statement is true, and in fact for any $c\geq 0$ and $\epsilon>0$ the number of $n$ such that $p_n\leq x$ and $d_{n+1}\geq cd_n$ is\[\frac{\pi(x)}{c+1}+O\left(\frac{x}{(\log x)^{3/2+\epsilon}}\right).\]

  • So are there any rules to accepting conditional solutions for problems reduced to well-known conjectures? I followed my (GPT assisted) work above with a uniform Hardy--Littlewood prime $k$-tuple conjecture obtain the solution. Obviously no one is solving that any time soon, so where would this go?

    Uniform Hardy--Littlewood correlations (conjectural).
    Fix $k \ge 1$ and $\lambda > 0$. Uniformly for distinct integers\[
    0 \le h_1 < h_2 < \cdots < h_k \le \lambda \log x,
    \]one has\[
    \sum_{n \le x} \prod_{j=1}^k \Lambda(n+h_j)
    =
    \bigl(\mathfrak{S}(\{h_1,\dots,h_k\}) + o_{k,\lambda}(1)\bigr)\,x,
    \]where $\Lambda$ is the von Mangoldt function and
    $\mathfrak{S}(\{h_1,\dots,h_k\})$ is the Hardy--Littlewood singular series.

    Assume this conjecture holds for all $k \ge 1$.
    Fix $\lambda > 0$. For $x$ large, form the scaled point set\[
    \mathcal{P}_x
    =
    \left\{
    \frac{p-x}{\log x} \in [0,\lambda]
    :\ p \text{ prime},\ x < p \le x + \lambda \log x
    \right\}.
    \]Then $\mathcal{P}_x$ converges in finite-dimensional distributions
    to a rate-$1$ Poisson point process on $[0,\lambda]$.

    Let\[
    0 \le a_1 < b_1 \le \cdots \le a_m < b_m \le \lambda,
    \]and let $N_i(x)$ count primes in the interval\[
    (x + a_i \log x,\ x + b_i \log x].
    \]Joint convergence to independent Poisson random variables with means
    $(b_i - a_i)$ follows by computing mixed factorial moments\[
    \mathbb{E}\!\left[\prod_i (N_i)_{r_i}\right]
    \]averaged over $x$.
    Expanding these moments reduces to sums of the above type with
    $k = \sum r_i$, where the shifts $h_j$ range over a set of size
    $O(\log x)$.
    The conjectural asymptotic supplies the main term, and the remaining
    singular-series averages match those of a Poisson process.

    Let $(T_0 < T_1 < T_2 < \cdots)$ be the points of a rate-$1$ Poisson point
    process on $\mathbb{R}$, conditioned to have a point at $0$.
    Then the successive gaps\[
    G_i = T_{i+1} - T_i
    \]are independent $\mathrm{Exp}(1)$ random variables.

    Let $X$ and $Y$ be independent $\mathrm{Exp}(1)$ random variables.
    For each $c > 0$,\[
    \mathbb{P}(Y \ge cX) = \frac{1}{1+c}.
    \]In particular,\[
    \mathbb{P}(Y > X) = \mathbb{P}(Y < X) = \tfrac12,
    \qquad
    \mathbb{P}(Y = X) = 0.
    \]Indeed,\[
    \mathbb{P}(Y \ge cX)
    =
    \int_0^\infty \mathbb{P}(Y \ge cx)\,e^{-x}\,dx
    =
    \int_0^\infty e^{-cx}e^{-x}\,dx
    =
    \frac{1}{1+c}.
    \]Assume the uniform Hardy--Littlewood conjecture above.
    Choose a prime $p_n$ uniformly at random among primes $\le x$ and consider
    the local prime configuration in the interval\[
    [p_n,\ p_n + \lambda \log p_n].
    \]After rescaling by $\log p_n$, the consecutive gaps\[
    \left(
    \frac{d_n}{\log p_n},
    \frac{d_{n+1}}{\log p_n}
    \right)
    \]converge in distribution to independent $\mathrm{Exp}(1)$ random variables
    $(X,Y)$.
    Therefore, for each fixed $c > 0$, the limiting density exists and equals\[
    \delta(\{n : d_{n+1} \ge c\,d_n\})
    =
    \mathbb{P}(Y \ge cX)
    =
    \frac{1}{1+c}.
    \]Taking $c = 1$ gives density $1/2$.
    Since ties have density zero, the same limit holds with $\ge$ replaced by
    $>$, and similarly for $<$.

          • So to review the bidding of a conditional resolution:

            1) We show that ($d_n = d_{n+1} \le y$) and ($d_n = d_{n+1} > y$) have density zero:

            The argument partitions at $y = (\log x)^{4/3}$.
            Small equal gaps ($d_n = d_{n+1} \le y$) are bounded by uniform Brun sieve estimates for $3$-tuples.
            Large equal gaps ($d_n = d_{n+1} > y$) are bounded using the telescoping sum
            $$\sum_{m \le \pi(x)} d_m = p_{\pi(x)+1} - 2 \ll x,$$
            which implies that the number of gaps exceeding $y$ is $\ll x/y$.
            With the choice $y = (\log x)^{4/3}$, both contributions are $o(\pi(x))$, so
            $$T(x)/\pi(x) \to 0.$$

            2) We follow Banks [Ba23] who gives a heuristic argument, assuming a quantitative form of the prime tuples conjecture, that the first statement is true, and in fact for any $c\geq 0$ and $\epsilon>0$ the number of $n$ such that $p_n\leq x$ and $d_{n+1}\geq cd_n$ is\[\frac{\pi(x)}{c+1}+O\left(\frac{x}{(\log x)^{3/2+\epsilon}}\right).\]

  • Followed Terry Tao's suggestion above to use sieve theory methods.

    After several iterations (and outstanding help from Kimi K2, Grok, and Gemini) on the optimal splitting threshold, the proof is now complete: equal consecutive prime gaps have asymptotic density zero, thereby reducing Erdős Problem #218 to showing that each strict inequality $d_{n+1} > d_n$ and $d_{n+1} < d_n$ has density $1/2$.

    The argument partitions at $y = (\log x)^{4/3}$.
    Small equal gaps ($d_n = d_{n+1} \le y$) are bounded by uniform Brun sieve estimates for $3$-tuples.
    Large equal gaps ($d_n = d_{n+1} > y$) are bounded using the telescoping sum
    $$\sum_{m \le \pi(x)} d_m = p_{\pi(x)+1} - 2 \ll x,$$
    which implies that the number of gaps exceeding $y$ is $\ll x/y$.
    With the choice $y = (\log x)^{4/3}$, both contributions are $o(\pi(x))$, so
    $$T(x)/\pi(x) \to 0.$$
