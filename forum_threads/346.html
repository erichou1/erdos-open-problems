OPEN This is open, and cannot be resolved with a finite computation.
Let $A=\{1\leq a_1< a_2<\cdots\}$ be a set of integers such that
  • $A\backslash B$ is complete for any finite subset $B$ and
  • $A\backslash B$ is not complete for any infinite subset $B$.
(Here 'complete' means all sufficiently large integers can be written as a sum of distinct members of the sequence.)

Is it true that if $a_{n+1}/a_n \geq 1+\epsilon$ for some $\epsilon>0$ and all $n$ then\[\lim_n \frac{a_{n+1}}{a_n}=\frac{1+\sqrt{5}}{2}?\]
#346: [ErGr80,p.57]
number theory | complete sequences

Comment activity that has not yet been incorporated into the remarks
There are no solutions, partial or complete, claimed in the comments.
Graham [Gr64d] has shown that the sequence $a_n=F_n-(-1)^{n}$, where $F_n$ is the $n$th Fibonacci number, has these properties. Erdős and Graham [ErGr80] remark that it is easy to see that if $a_{n+1}/a_n>\frac{1+\sqrt{5}}{2}$ then the second property is automatically satisfied, and that it is not hard to construct very irregular sequences satisfying both properties.

        That said, what Price constructs is not a cheap pathological sequence either. This Burr-Erdos' 1981 paper studies completeness of "perturbed" sequences and has results similar to Price's. It's similar enough that Price should cite this paper as a precursor of ideas, although the paper's results do not by itself give Price's results. This shows that such constructions are of interest and Price can be regarded as solving a variant problem.

        Also, from Burr-Erdos we also know that if the limit exists, it cannot be greater than $\varphi$. The remaining case where the limit is less than $\varphi$ is probably still open.

        • I think there are two possible interpretations of Problem 346.
          $\textbf{First, convergence-from-hypotheses interpretation:}$ The original hypotheses are asked to imply both that the quotient sequence $a_{n+1}/a_n$ converges and that its limit is $\varphi$.
          Liam Price's answer appears to address this interpretation.
          $\textbf{Second, limit-exists interpretation:}$ The quotient limit is assumed to exist, and the question is whether the hypotheses force that limit to be $\varphi$.
          Dogmachine suggested this as a possible intended reading.

          For the $\textbf{second, limit-exists interpretation}$, I have an attempted Lean 4 formalization giving an $\textbf{affirmative answer}$. In informal terms, the formalized statement is: If the two deletion hypotheses hold, the sequence has a uniform ratio gap, and there exists $L>1$ such that $a_{n+1}/a_n \to L$, then $a_{n+1}/a_n \to \varphi$.

          • Congrats.

            This Lean proof correctly shows that $\lim_{n \to \infty} a_{n+1}/a_n = L \neq \varphi$ is impossible. The case $L > \varphi$ is handled in Burr-Erdos as above. It remains to rule out the case $L < \varphi$.

            Choose $L < \rho < \varphi$. Eventually $a_{n+1}/a_n < \rho$. The key is
            $$ \sum_{j=2}^\infty \rho^{-j} = \frac{1}{\rho(\rho - 1)} > 1. $$
            Choose $m \ge 1$ so that $\sum_{j=2}^{m+1} \rho^{-j} > 1$. So eventually,
            $$ a_{n+1} < a_{n-m} + \dots + a_{n-1}. $$

            Choose $H$ so every integer at least $H$ is representable. There is $M$ large enough so that $a_1, \dots, a_M$ can represent every integer in the interval $[H, T]$ where $T = 2a_M - 1$. (Proof: for $M$ large, $a_M + H < a_{M+1}$. So if $H \leq t \leq a_M + H$, $t$ can be represented by terms up to $a_M$. Otherwise if $a_M + H < t < 2a_M$, $t - a_M$ can be represented by terms up to $a_{M-1}$, and then we add $a_M$.)

            We now delete the indices
            $$ D = \{ M + m + 1 + k(m + 2): k \ge 0 \} $$
            and show the resulting sequence is still complete.

            $\textbf{Interval lemma}$. If a collection of terms can represent the interval $[a, b]$, and a new term $x \leq b - a + 1$, then the old terms and $x$ together can represent the interval $[a, b + x]$. (Proof: if $b < t \leq b + x$, represent $t - x$ and add $x$.)

            We can show inductively that, for $n \ge M$, the terms $a_i$ for $i \leq n$ and $i \notin D$ can represent the interval $[H, T + \sum_{M < i \leq n, i \notin D} a_i]$. This is by using the interval lemma and the key inequality stated above. I think there may be some technicalities, which can solved by making $M$ larger if necessary.
