Is there a polynomial $f:\mathbb{Z}\to \mathbb{Z}$ of degree at least $2$ and a set $A\subset \mathbb{Z}$ such that for any $n\in \mathbb{Z}$ there is exactly one $a\in A$ and $b\in \{ f(k) : k\in\mathbb{Z}\}$ such that $n=a+b$?
#477: [ErGr80,p.95]

A question of Erdős and Graham, who thought the answer was negative. Clearly any such $A$ must be infinite.

A simple proof that this is impossible for $f$ of degree $2$ is given in the comments (with the combined efforts of AlphaProof and Adenwalla): let $f(x)=c_2x^2+c_1x+c_0$ with $c_1\neq 0$. For any infinite $A$ there exist $a,b\in A$ such that $a-b=2c_1k$ for some $k\geq 1$, whence\[0\neq a-b=f(k)-f(-k)\]and so $n=a+f(-k)=b+f(k)$ has two distinct solutions. If $c_1=0$ then we can argue similarly finding $a,b\in A$ with $a-b=4c_2k$ for some $k\geq 1$, whence\[0\neq a-b=f(k+1)-f(k-1).\]The same argument works for any $f$ such that $f(\mathbb{Z})-f(\mathbb{Z})$ contains all multiples of some fixed $q\geq 1$, such as any polynomial of the shape\[f(x)=g((x-k)^2)+c_1x+c_0\]for some $g\in \mathbb{Z}[x]$ and $k,c_1,c_0\in \mathbb{Z}$ with $c_1\neq 0$.

  • AlphaProof found a counterexample to the previous formulation of this problem.This was an formal version, which can be summarised as follows:

    1. Pick $f = X^2 - X + 1$
    2. Now $A$ can't be finite.
    3. If $A$ is infinite there are two elements $a, b$ which differ by a positive multiple of $4$ (say $a = b + 4x$)
    4. Now consider the pairs $(a, f(2x))$ and $(b, f(2x + 1))$. The point being that $f(2x + 1) - f(2x) = 4x$, so these give two representations of the same number.

    This proof can be generalised to all polynomials of degree $2$ of the form $ax^2 + bx + c$ where $a | b$ and $b ≠ 0$.

    Formalised statement and proof (cleaned up and generalised from the AlphaProof proof) see here:
    https://github.com/google-deepmind/formal-conjectures/blob/24a4308a38b97e3e5b7da6ccff57157de432a1c1/FormalConjectures/ErdosProblems/477.lean#L44

    So this gives an infinite family of degree $2$ polynomials for which no such $A$ exists.

    [ErGr80] mentions

    [Sek59](http://dml.cz/dmlcz/100376) Milan Sekanina, Замечания к фактoризации беcкoнечнoй цикличеcкoй группы, Czechoslovak Mathematical Journal, Vol. 9 (1959), No. 4, 485–495

    and if I understand correctly the $X^2$ counterexample is also mentioned on page 490 there. And then the question is asked for other monomials $X^k$ for $k\ge3$.

    We added those variants to formal-conjectures: https://github.com/google-deepmind/formal-conjectures/pull/1510/

    • This can be generalised even when $b=0$. It suffices to show that for any monic quadratic polynomial $f:\mathbb{Z}\rightarrow\mathbb{Z}$, there exists an integer $n$ such that for all integers $k$, there is $x,y\in\mathbb{Z}$ such that $f(x)-f(y)=kn$. That is, all multiples of $n$ are in the difference set of $f(x)$. This is because any set $A$ must be infinite, and so $A-A$ must contain some multiple of $n$.

      Let $f(x)=ax^2+bx+c$ with $a,b,c\in\mathbb{Z}$. Note that if all multiples of $n$ are in the difference set of $f(x)$, then for any $d\in\mathbb{Z}$, all multiples of $n$ are in the difference set of $f(x)+d$.

      It has been shown that $x^2$ and $x^2-x+1$ each have difference sets that contain all multiples of $4$. The same then holds true for $(x+k)^2$ and $(x+k)^2-(x+k)+1$ for any $k\in\mathbb{Z}$. As $(x+k)^2=x^2+2kx+k^2$ and $(x+k)^2-(x+k)+1=x^2+(2k-1)x+k^2-k+1$, we have covered $x^2+bx+c$ for any $b,c\in\mathbb{Z}$ (that is, the difference set of any polynomial of this form contains all multiples of $4$).

      Then, for any $a\in\mathbb{Q}$ and monic quadratic $f:\mathbb{Z}\rightarrow\mathbb{Z}$ such that $a\times f:\mathbb{Z}\rightarrow\mathbb{Z}$, all multiples of $4a$ are in the difference set of $a\times f(x)$. Therefore any polynomial of the form $ax^2+bx+c$ where $a|b$ contains all multiples of $4a$ in its difference set.

      • This can actually be generalised for all quadratic polynomials $f:\mathbb{Z}\rightarrow\mathbb{Z}$. Consider $f(x)=ax^2+bx+c$. Then the difference set of $f(x)$ contains all terms of the form $$f(x+2l)-f(x)=(4ax+4al+2b)l.$$ As $x$ can be any integer, this is all terms of the form $$4al^2+2bl\mod 4al\equiv 2bl\mod 4al.$$ As we can vary $l$ to be any integer, the family of congruences $\{2bl\mod 4al\mid l\in\mathbb{Z}\}$ covers all multiples of $4a+2b$ (if $b\neq 0$, we can take multiples of $2b$).

        EDIT: A simpler way to see this is to take $f(x)=ax^2+bx+c$ with $b\neq0$ and consider $f(k)-f(-k)=2bk$ for any $k\in\mathbb{Z}$. So all multiples of $2b$ are in the difference set of $f(x)$.

        If $b=0$ then $f(x+1)-f(x-1)=4ax$, so all multiples of $4a$ are in the difference set of $f(x)$.

        Then for any infinite $A\subset\mathbb{Z}$, there must be some multiple of $2b$ and some multiple of $4a$ in $A-A$.

  • Take $f(x) =x^2$. Then for any $a, b\in A$ and integers $m, n$, we want to avoid $a-b=m^2-n^2$ unless $a=b$. All odd numbers $>1$ and multiples of 4 can be expressed as a difference of two positive squares. So $A-A$ can only contain terms that are $2\mod 4$ or $0, 1$.

    Then letting $|A|\geq 5 $, there must be and $a, b, c, d, e\in A$ such that $a<b<c<d<e$. Then no two of $a, c, e$ have difference $1$ or $0$. So we must have $a-b\equiv b-c\equiv a-c \equiv 2\mod 4$. But $$a-c=(a-b) +(b-c) \equiv 2+2\equiv 0\mod 4. $$
    So $|A|\leq 4$, but this is clearly not enough to represent all integers.

    • The claim is also trivially false for any $f$ that is non-injective on the natural numbers, such as $f(x) = (x-1)^2$. (EDIT: I see the formalized statement in Lean avoids this loophole by enumerating using $(a,f(n))$ rather than $(a,n)$.) In order to avoid such trivialities, I would imagine the question should be interpreted in the sense indicated in the remarks, that no $A$ exists for any $f$.

      EDIT: On checking [ErGr80, p 95-96] it appears that (a) one is indeed enumerating $(a,f(n))$ rather than $(a,n)$, and (b) the problem should be understood in the strong sense that no such $A$ exists for any polynomial $f$ of degree at least two.
