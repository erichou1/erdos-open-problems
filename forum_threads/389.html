Is it true that for every $n\geq 1$ there is a $k$ such that\[n(n+1)\cdots(n+k-1)\mid (n+k)\cdots (n+2k-1)?\]
#389: [ErGr80,p.75]
number theory

Asked by Erdős and Straus.
For example when $n=2$ we have $k=5$:\[2\times 3 \times 4 \times 5\times 6 \mid 7 \times 8 \times 9\times 10\times 11.\]and when $n=3$ we have $k=4$:\[3\times 4\times 5\times 6 \mid 7\times 8\times 9\times 10.\]Bhavik Mehta has computed the minimal such $k$ for $1\leq n\leq 18$ (now available as A375071 on the OEIS).

  • The similarity with #396 has already been noted. If one makes the change of variables $(n',k') = (n+k-1, n)$, the similarity becomes more obvious: removing the primes, the question now asks whether, for every $k$, there exists $n$ such that
    $$ \prod_{i=0}^k (2n-i) | (k-1)! \binom{2n}{n}.$$

  • Computational note for [389]. For $n=25$, I searched for a $k\ge 1$ such that\[
    25\cdot 26\cdots (25+k-1)\ \mid\ (25+k)(25+k+1)\cdots (25+2k-1).
    \]I checked all $1\le k\le 12{,}000{,}000$ and found no solution in this range, hence\[
    k_{\min}(25) > 12{,}000{,}000.
    \](Of course this does \emph{not} prove nonexistence of such a $k$ for $n=25$; it is only a lower bound.)

    Method: track prime-adic valuation differences\[
    d_p(k)=v_p\!\Big(\prod_{j=0}^{k-1}(n+k+j)\Big)-v_p\!\Big(\prod_{j=0}^{k-1}(n+j)\Big),
    \]and update incrementally via\[
    d_p(k+1)=d_p(k)+v_p(n+2k)+v_p(n+2k+1)-2v_p(n+k),
    \]factoring each integer using a precomputed smallest-prime-factor table up to $n+2k_{\max}+1$.
    As a sanity check, the implementation reproduces the sample solutions $n=2\Rightarrow k=5$ and
    $n=3\Rightarrow k=4$, and agrees with the known value $n=4\Rightarrow k=207$ (consistent with
    OEIS A375071, noting the index shift).

  • Let us call a pair (n,k) ES-nice if the divisibility condition holds for it. Looking at A375071 and doing some numerical computations leads me to the conjecture:

    (2m+1,k) is ES-nice, if (2m,k+1) is ES-nice.
    If this is true, the restricted version of problem 389 to odd numbers n = 2m+1 might be somewhat simpler than 389 itself.

    The converse direction holds often but not always.

    • I didnot see any reason why this conjecture should be true.
      You have divisibilities with a lot of common terms, so it is not surprising that it often works.
      Basically one wants that $2m+2k+1$ has a crucial prime factor of $\prod_{i=1}^k (2m+i)$,
      where crucial means that it did not appear in $\prod_{i=1}^k (2m+k+i)$ with sufficiently high multiplicity. That happens rarely, but there are cases.
      E.g. when $n=4$ and $k=987$ or $3993$,
      or $n=6$ and $k=19500$.
      So conjecture refuted.
