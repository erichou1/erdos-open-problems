Let $f(N)$ be the largest size of $A\subseteq \{1,\ldots,N\}$ with the property that there are no distinct $a,b,c\in A$ such that\[[a,b]=[b,c]=[a,c],\]where $[a,b]$ denotes the least common multiple.

Estimate $f(N)$ - in particular, is it true that $f(N)=o(N)$?
#536: [Er64,p.646][Er70,p.124][Er73,p.124]
number theory

The corresponding quantity is $\gg N$ if we ask for four elements with the same pairwise least common multiple, as shown by Erdős [Er62] (with a proof given in [Er70]).

This was also asked by Erdős at the 1991 problem session of West Coast Number Theory.

Abbott and Gardner [AbGa67] proved\[f(N) \geq (1-o(1))(\log\log N)\frac{N}{\log N}.\]In the comments Weisenberg sketches a proof that, for some function $\omega(N)\to \infty$ as $N\to \infty$,\[(\log\log N)^{\omega(N)}\frac{N}{\log N}\ll f(N) \leq \left(\frac{221}{225}+o(1)\right)N.\]See also [535], [537], and [856]. A related combinatorial problem is asked at [857].

  • I made a Lean formalization attempt for the following upper bound in Erdos Problems #536: $|A| \le N-\lfloor N/6 \rfloor$
    Equivalently, this is $|A| \le \lceil 5N/6\rceil$; asymptotically, the coefficient is $5/6 \approx 0.833333$, i.e. about $0.833333N$.
    Here $A \subseteq \{1,\ldots,N\}$ has no three distinct elements $a,b,c$ such that $\operatorname{lcm}(a,b)=\operatorname{lcm}(a,c)=\operatorname{lcm}(b,c)$.

    • I have run a screening check and the Lean is correct.

      This is the strongest upper bound so far, and the argument is surprisingly simple given the efforts already spent by others.

      Write an integer $n = m 2^i 3^j$ where $(m, 6) = 1$. For a fixed $m$, identify $n$ with a point $(i, j)$ in a lattice.

      The condition on $A$ implies that the 3 points which form a "corner": $\{ (i, j), (i - a, j), (i, j - b) \}$, $a, b > 0$, cannot all be in $A$.

      Define a projection to the axes
      $$ \pi(i, j) = \begin{cases} (i, j), & i = 0\text{ or }j = 0, \\
      (i, 0), & \text{some point of }A\text{ lies to its left}, \\
      (0, j), & \text{otherwise}.
      \end{cases}$$
      It is easy to check that the projection $\pi$ is injective by the corner observation above.

      It immediately follows:
      $$ |A| \le \#\{\text{axis points}\} = \{ n \le N: 6 \nmid n \} = N - \lfloor N/6 \rfloor. $$

      This kind of forbidden pattern is known in literature as $\Gamma$-free $0$-$1$ matrices. See e.g. this 2018 paper by Bényi and Nagy. It is known that such a matrix of size $n \times k$ has at most $n + k - 1$ $1$'s, which also follows from the above projection argument.

      Nevertheless, the consideration of this specific lattice for this problem could be novel.

  • I plan to post a note to arxiv later showing that $f(N) \ge \frac{N}{(\log N)^{0.059... + o(1)}}$ with a more involved argument, but for now here's a quick sketch that $f(N) \ge \frac{N}{(\log N)^{1 - 1/e + o(1)}}$:

    Let\[
    L=\log\log N,
    \qquad
    k=\left\lfloor \frac{L}{e}\right\rfloor.
    \]Let $X$ be the set of squarefree $n\in(N/2,N]$ with $\omega(n)=k$ and let $J(\alpha) = 1 - \alpha + \alpha\log\alpha$. The squarefree version of the Sathe--Selberg theorem gives, uniformly for fixed $0<\alpha<1$,\[
    \#\{n\in(N/2,N]:n\text{ squarefree},\ \omega(n)=\lfloor \alpha\log\log N\rfloor\}
    =\frac{N}{(\log N)^{J(\alpha)+o(1)}}.
    \]Taking $\alpha=1/e$ gives\[
    |X|=\frac{N}{(\log N)^{J(1/e)+o(1)}}.
    \]Color every prime independently and uniformly with one of $k$ colors. For a coloring $\chi$, let $A_\chi\subseteq X$ consist of those $n$ whose $k$ prime factors receive all $k$ colors exactly once. For fixed $n\in X$,\[
    \mathbb{P}(n\in A_\chi)=\frac{k!}{k^k}=\exp(-k+o(k))=(\log N)^{-1/e+o(1)}.
    \]Hence\[
    \mathbb{E}_\chi \left[|A_\chi|\right]=\frac{N}{(\log N)^{J(1/e)+1/e+o(1)}}
    =\frac{N}{(\log N)^{1-1/e+o(1)}}.
    \]It is easy to see that $A_\chi$ is LCM-triangle-free, so we're done.

  • Hey everyone, I wanted to share some recent progress I've made on upper-bounding the density of sets avoiding these "bad triples" (where $\operatorname{lcm}(a,b) = \operatorname{lcm}(b,c) = \operatorname{lcm}(a,c)$).

    I've been tackling this by constructing explicit, asymptotically dense packings of bad triples to bound the complement set. The main idea is to classify integers by their "signatures" - specifically, their $p$-adic valuations modulo some cutoff $K_p$ for a small set of primes, like $\mathcal{P} = \{2, 3, 5, 7, 11\}$.

    By looking at different triples of coprime coefficients $(A, B, C)$ (representing $(a, b, c) = (ABm, ACm, BCm)$) and assigning them to these valuation classes, we can calculate the natural density of each class and frame the whole thing as a massive Set Packing problem.

    Here are the best bounds I've managed to find so far:

    1. Exact Disjoint Matching
    By finding a strictly element-disjoint collection of bad-triple families and converting the problem to solving an Integer Linear Program over the signature classes, I've brought the upper bound down to:
    $$f(N) \leq \left(\frac{203{,}447}{230{,}400} + o(1)\right )N \approx 0.88302N.$$
    The construction here is a set of $42$ triples and seems to be optimal for $K = (4, 4, 2, 2, 2)$.

    2. Fractional Matching (LP)
    By taking the continuous LP relaxation of this matching and using LP duality, we can improve on this method. I wrote a script to compute a floating-point optimum, identify the binding constraints, and then solve the resulting system exactly over $\mathbb{Q}$ using fraction-free Gaussian elimination.

    $$f(N) \leq \left (\frac{151{,}594{,}752{,}885{,}431{,}136{,}780{,}159{,}002{,}252{,}347}{172{,}688{,}901{,}829{,}024{,}375{,}189{,}779{,}589{,}939{,}200} + o(1)\right )N \approx 0.877849 N.$$
    Because this is extracted as an exact rational vertex of the LP and verified over $\mathbb{Q}$, there are no floating-point artifacts, though there should exist better solutions.

  • I think the elementary packing bound recorded here can be improved from
    $f(N)\le \left(\frac{221}{225}+o(1)\right)N$
    to
    $f(N)\le \left(\frac{43}{48}+o(1)\right)N$.

    The construction uses three pairwise disjoint families of lcm-triangles:
    $\{2m,3m,6m\}$, $\{4m,5m,20m\}$, and $\{7m,9m,63m\}$,
    with valuation restrictions chosen to separate all generated triples.

    For the family $\{2m,3m,6m\}$, take
    $6m\le N$ and $v_2(m)\equiv v_3(m)\equiv0\pmod 2$.
    This contributes
    $\frac{N}{6}\cdot\frac{2}{3}\cdot\frac{3}{4}+o(N)=\frac{N}{12}+o(N)$.

    For the family $\{4m,5m,20m\}$, take
    $20m\le N$,
    $v_2(m)\equiv0\pmod 4$, and
    $v_3(m)\equiv v_5(m)\equiv0\pmod 2$.
    This contributes
    $\frac{N}{20}\cdot\frac{8}{15}\cdot\frac{3}{4}\cdot\frac{5}{6}+o(N)
    =\frac{N}{60}+o(N)$.

    For the family $\{7m,9m,63m\}$, take
    $63m\le N$,
    $v_2(m)\equiv0\pmod 4$,
    $v_3(m)\equiv0\pmod 4$, and
    $v_5(m)\equiv v_7(m)\equiv0\pmod 2$.
    This contributes
    $\frac{N}{63}\cdot\frac{8}{15}\cdot\frac{27}{40}\cdot\frac{5}{6}\cdot\frac{7}{8}+o(N)
    =\frac{N}{240}+o(N)$.

    Each template is an lcm-triangle:
    $[2m,3m]=[2m,6m]=[3m,6m]=6m$,
    $[4m,5m]=[4m,20m]=[5m,20m]=20m$, and
    $[7m,9m]=[7m,63m]=[9m,63m]=63m$.

    The valuation signatures separate the three roles inside each layer and also separate the layers from one another, so the constructed lcm-triangles are pairwise disjoint. Hence the total number of disjoint forbidden triples is
    $\frac{N}{12}+\frac{N}{60}+\frac{N}{240}+o(N)=\frac{5N}{48}+o(N)$.

    Therefore any lcm-triangle-free subset of $[N]$ must omit at least one element from each of these triples, giving
    $f(N)\le N-\frac{5N}{48}+o(N)=\left(\frac{43}{48}+o(1)\right)N$.

    This is still only a constant-density packing improvement, not progress all the way to the $o(N)$ question: any proof using only pairwise disjoint forbidden triples cannot force an upper bound below $2N/3$.

    For the full paper, here it is
    https://pdfhost.io/v/NnHQ32EvjQ_Erdos_536_Partial_Progress

    • The remaining two page numbers are [Er70,p.124][Er73,p.124].

      The claim that if $A$ has no lcm triangles then $|A| \leq cN$ for some $c < 1$ for all large $N$ can actually be proven by purely elementary means. We do so as follows: observe that as $m$ ranges over the integers from 1 to $\lfloor\frac{N}{15}\rfloor$ with $\text{gcd}(m, 30) = 1$, the sets $\{6m, 10m, 15m\}$ are disjoint lcm triangles contained in $\{1, \dots, N\}$. (The condition $\text{gcd}(m, 30) = 1$ isn't actually necessary to make these sets lcm triangles, but it is necessary for disjointness.) The amount of such $m$ is asymptotically $\frac{1}{15} \cdot \frac{\phi(30)}{30} \cdot N$, and $\phi(30) = 8$, so this is $\frac{4}{225}N$. If $A$ is lcm triangle-free, then at least one element must be removed for every such $m$, so we have $|A| \leq (\frac{221}{225} + o(1))N$.

      This proves the desired claim, but I have no doubt that it can be improved significantly. For example, I imagine we could include $m$ such that $\text{gcd}(m, 30) > 1$ as well as apply the same analysis to other lcm triangles $\{a, b, c\}$ with $\text{gcd}(a, b, c) = 1$ at the same time (as opposed to just $\{6, 10, 15\}$). The inclusion-exclusion principle might be all we need to do both of these things.

  • For the record, both times Erdős asked this question, he specified that $a,b,c$ are distinct - that may be worth clarifying here. It's obvious that allowing $a = b = c$ leaves us stuck with the empty set, so that's trivial, but allowing two of them to be equal is equivalent to requiring that $A$ be a primitive set, which is an interesting variant, but not what he asks.


    For convenience, call a set of three distinct numbers $\{a,b,c\}$ an lcm triangle if $[a,b] = [b,c] = [a,c]$. Then this problem asks how large a subset of $\{1,...,N\}$ can be if it contains no lcm triangles.


    Fix a positive integer $k$ and let $S_k$ be the set of all numbers of the form $p_1 \cdots p_k$, where each $p_i$ is prime and $\pi(p_i) \equiv i \pmod{k}$ for all $i$ (but the primes are not necessarily in increasing order). It's not hard to verify that $S_k$ contains no lcm triangles. Furthermore, it's not hard to verify that $S_k$ has positive relative lower density within the set of all $k$-almost primes, and it's known that the amount of $k$-almost primes up to $N$ is $\sim_k \frac{N (\log \log N)^{k-1}}{(k - 1)! \log N}$. As such, for each $N$, we can choose the value of $k$ that maximizes the size of $S_k \cap \{1,...,N\}$. By doing so, we see that there is always an lcm triangle-free set $A \subseteq \{1,...,N\}$ such that
    \begin{equation*}
    |A| \gg_t \frac{N (\log \log N)^t}{\log N}.
    \end{equation*}
    With a bit of work, I'm sure one could make this lower bound a little more explicit.
