Let $k\geq 3$ and $f(k)$ be the supremum of $\sum_{n\in A}\frac{1}{n}$ as $A$ ranges over all sets of positive integers which do not contain a $k$-term arithmetic progression. Estimate $f(k)$.

Is\[\lim_{k\to \infty}\frac{f(k)}{\log W(k)}=\infty\]where $W(k)$ is the van der Waerden number?
#169: [Er77c][ErGr79][Er80,p.92][ErGr80]
additive combinatorics | arithmetic progressions

Comment activity that has not yet been incorporated into the remarks
There are no solutions, partial or complete, claimed in the comments.
Berlekamp [Be68] proved $f(k) \geq \frac{\log 2}{2}k$. Gerver [Ge77] proved\[f(k) \geq (1-o(1))k\log k.\]It is trivial that\[\frac{f(k)}{\log W(k)}\geq \frac{1}{2},\]but improving the right-hand side to any constant $>1/2$ is open.

Gerver also proved (see the comments for an alternative argument of Tao) that [3] is equivalent to $f(k)$ being finite for all $k$.

The current record for $f(3)$ is $f(3)\geq 3.00849$, due to Wróblewski [Wr84]. Walker [Wa25] proved $f(4)\geq 4.43975$.

Walker [Wa25] has shown that it suffices to consider Kempner sets (that is, sets of integers defined as all those whose base $b$ digits are contained in some $S\subset \{0,\ldots,b-1\}$ for fixed $b$ and $S$), in the sense that for any $k\geq 3$ and $\epsilon>0$ there is a Kempner set $A$ lacking $k$-term arithmetic progressions such that\[\sum_{n\in A}\frac{1}{n}\geq f(k)-\epsilon.\]In [Er80] Erdős asks whether for every $\epsilon>0$ and $k\geq 3$ if $A$ is a set of integers without a $k$-term arithmetic progression such that $\min(A)$ is sufficiently large in terms of $\epsilon$ and $k$ then $\sum_{n\in A}\frac{1}{n}<\epsilon$.

    • I'm not seeing this. Of course, if [3] is false for $k$, then $f(k) = \infty$. But how do we prove the converse? That is, how do we rule out the case where
      \begin{equation*}
      \sum_{n \in A} \frac{1}{n} < \infty
      \end{equation*}
      for all $k$-AP-free sets $A$, but
      \begin{equation*}
      \sup_{\text{$A$ is $k$-AP-free}} \left(\sum_{n \in A} \frac{1}{n}\right) = \infty?
      \end{equation*}
      My first thought was to try proving that if there are $k$-AP-free sets $A_1, A_2,...$, whose respective sums of reciprocals approach infinity, then they can be used to construct a $k$-AP-free set $A$ whose sum of reciprocals actually diverges. One method is to take the compactness approach: pick some positive integer sequence $n_1 < n_2 < \cdots$, and pick some (maximal) subset of $\{1,...,n_1\}$ that appears in infinitely many $A_i$ and add it to $A$, then restrict the sequence of $A_i$s to the ones whose initial segment is the same as $A$ and then repeat with $n_2$, and so on. It's easy to prove that $A$ has no $k$-APs, but the sum of reciprocals of elements of $A$ does not necessarily diverge (in fact, if the smallest element of $A_i$ approaches infinity as $i$ does, then $A$ is the empty set). So compactness without any considerations specific to APs won't be enough to solve this problem. Did you have a different approach (or perhaps a way of improving this one) in mind?

      • If the supremum is finite, then recursively one can construct finite $k$-AP-free sets $A_1,A_2,A_3,\dots$ with $\sum_{n \in A_j} \frac{1}{n} \geq 1$ (say) where the minimal element of each $A_j$ is greater than twice the maximal element of $A_1 \cup \dots \cup A_{j-1}$ (by taking $A_j$ to be a $k$-AP-free set with extremely large sum of reciprocals and then deleting all elements less than twice that maximal element). Then, the $A_j$ are disjoint, and the union $\bigcup_{j=1}^\infty A_j$ will have divergent sum of reciprocals while still being $k$-AP-free.

        EDIT: okay, merely keeping each $A_j$ greater than twice the maximal element of the $A_1 \cup \dots \cup A_{j-1}$ isn't going to be enough, due to $k$-term arithmetic progressions that have one element in $A_1 \cup \dots \cup A_{j-1}$ and the other $k-1$ in $A_j$. But if one randomly refines each $A_j$ down to a proportion $\varepsilon$ of the original set, and then further removes any surviving $k$-term APs of this form (which removes $O_k(|A_1 \cup \dots \cup A_{j-1}| \varepsilon^{k-1})$ of the original elements of $A_j$, where we weight each natural number $n$ by $1/n$ for the purposes of measuring size), I think one can still end up with a reduced set $A'_j$ with $\sum_{n \in A'_j} \frac{1}{n}$ as large as one pleases (by first taking $\varepsilon$ small, and then using the hypothesis of infinite supremum), with the property that the inclusion of $A'_j$ to $A'_1 \cup \dots \cup A'_{j-1}$ does not add any new $k$-APs, and the argument proceeds as before.
